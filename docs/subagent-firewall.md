# Subagent UC-firewall

Operator runbook for the `audit-subagent-uc-firewall` CI gate (Lane B Task B-7).

## What this protects

Handwritten catalogue content lives in UC JSON sidecars under
`content/cat-*/UC-*.json`. Subagents may build tooling that *reads* sidecars
(audits, generators, MCP tools) and may change schemas, CI, frontend shells,
and tests — but they must not commit prose or field edits inside a UC sidecar.

See [parallel-execution-substrate.md §4](parallel-execution-substrate.md#4-subagent-uc-firewall)
for the full protected-path contract and lift-loop boundary.

## Label contract

| Label | Meaning |
| --- | --- |
| `subagent-authored` | PR was opened by the subagent dispatch protocol (§5.2). UC sidecar diffs are **blocked**. |
| `maintainer-authored` | Human or maintainer session. UC sidecar diffs are **allowed**. |
| *(no label)* | Treated as maintainer — UC sidecar diffs are **allowed**. |

The orchestrator applies `subagent-authored` when opening implementer PRs so CI
can enforce the firewall without guessing authorship from commit metadata.

## Positive cases

- **Maintainer PR** (no `subagent-authored` label) editing `content/cat-*/UC-*.json` → PASS
- **`maintainer-authored` label** with UC sidecar changes → PASS
- **`subagent-authored` PR** touching only code, docs, tests, CI, schemas → PASS

## Failing cases

- **`subagent-authored` PR** with one or more changed paths matching
  `content/cat-*/UC-*.json` → FAIL (blocks merge)

## Local repro

Hermetic evaluation without `gh` or `git`:

```bash
cd /path/to/repo
PYTHONPATH=src python3 -m splunk_uc audit-subagent-uc-firewall \
  --check \
  --labels subagent-authored \
  --paths content/cat-1-network/UC-1.1.1.json
# exit 1

PYTHONPATH=src python3 -m splunk_uc audit-subagent-uc-firewall \
  --check \
  --paths content/cat-1-network/UC-1.1.1.json
# exit 0 — no subagent label
```

Full CI invocation (as in `validate.yml`):

```bash
git fetch origin main --depth=50
export GH_TOKEN=…
export GITHUB_REPOSITORY=owner/repo
export GITHUB_EVENT_PULL_REQUEST_NUMBER=123
PYTHONPATH=src python3 -m splunk_uc audit-subagent-uc-firewall --check
```
