# UC-15.2.16 · Condensation Risk Alerts

## Description

Surface temp below dew point risks water on equipment and slip hazards near chilled doors.

## Value

Surface temp below dew point risks water on equipment and slip hazards near chilled doors.

## Implementation

Compute dew point from RH and air temp or ingest BMS dew point. Alert when within 1°C of condensation.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: RH + surface temp sensors.
• Ensure the following data sources are available: `sourcetype="sensor:condensation"` (surface_temp_c, dew_point_c).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Compute dew point from RH and air temp or ingest BMS dew point. Alert when within 1°C of condensation.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=environment sourcetype="sensor:condensation"
| where surface_temp_c <= dew_point_c + 1
| table _time, location, surface_temp_c, dew_point_c, rh_pct
```
