# UC-15.2.4 · Hot Aisle Temperature Trending

## Description

Hot aisle trends indicate cooling efficiency and capacity margin. Rising trends signal approaching cooling limits.

## Value

Hot aisle trends indicate cooling efficiency and capacity margin. Rising trends signal approaching cooling limits.

## Implementation

Deploy sensors in hot aisle containment. Track return air temperatures. Compare hot aisle temps across zones to identify cooling imbalances. Use prediction to forecast capacity issues.

## Detailed Implementation

Step 1 — Configure data collection
Deploy sensors in hot aisle containment. Track return air temperatures. Compare hot aisle temps across zones to identify cooling imbalances. Use prediction to forecast capacity issues.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=environment sourcetype="sensor:temperature" position="hot_aisle"
| timechart span=1h avg(temp_f) as avg_temp by zone
| predict avg_temp as predicted future_timespan=7
```
