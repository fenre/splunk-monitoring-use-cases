# UC-15.1.21 · Electrical Panel Phase Balancing

## Description

Panel-level phase balance differs from rack PDU balance; both matter for neutral harmonics and breaker life.

## Value

Panel-level phase balance differs from rack PDU balance; both matter for neutral harmonics and breaker life.

## Implementation

Alert when any phase deviates >20% from mean. Schedule rebalancing with facilities during maintenance.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Main distribution panel meters.
• Ensure the following data sources are available: `sourcetype="panel:phase"` (main_l1_a, main_l2_a, main_l3_a).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Alert when any phase deviates >20% from mean. Schedule rebalancing with facilities during maintenance.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=power sourcetype="panel:phase" panel_type="main"
| eval avg_a=(main_l1_a+main_l2_a+main_l3_a)/3
| eval dev=max(abs(main_l1_a-avg_a), abs(main_l2_a-avg_a), abs(main_l3_a-avg_a))
| where dev > avg_a*0.2
| table panel_id, main_l1_a, main_l2_a, main_l3_a, dev
```
