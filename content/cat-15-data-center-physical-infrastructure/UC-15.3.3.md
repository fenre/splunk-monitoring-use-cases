# UC-15.3.3 · Tailgating Detection

## Description

Tailgating bypasses access control, allowing unauthorized entry. Detection supports physical security integrity.

## Value

Tailgating bypasses access control, allowing unauthorized entry. Detection supports physical security integrity.

## Implementation

Analyze badge-in/badge-out patterns. Detect multiple entries without corresponding exits (or vice versa). Alert on anti-passback violations. Correlate with camera footage for investigation. Report on tailgating trends.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=physical sourcetype="access_control" door="DC_Main_Entry"
| transaction badge_id maxspan=10s
| where eventcount > 1 AND action="entry"
| table _time, badge_holder, badge_id, eventcount
```
