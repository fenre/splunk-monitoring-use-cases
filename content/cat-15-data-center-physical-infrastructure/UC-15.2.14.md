# UC-15.2.14 · Cooling Capacity vs IT Load

## Description

Compares available tons to IT heat load (from power meters) to expose margin before next summer.

## Value

Compares available tons to IT heat load (from power meters) to expose margin before next summer.

## Implementation

Convert IT kW to tons (approx). Alert when margin <15% of peak load forecast. Feed capacity planning.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: BMS + IT power feeds.
• Ensure the following data sources are available: `sourcetype="bms:cooling_capacity"` `sourcetype="power:it_heat"`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Convert IT kW to tons (approx). Alert when margin <15% of peak load forecast. Feed capacity planning.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cooling sourcetype="bms:cooling_capacity"
| eval it_heat_tons=it_load_kw * 0.284345
| eval margin_tons=cooling_capacity_tons - cooling_output_tons
| where margin_tons < it_heat_tons * 0.15
| table zone, cooling_capacity_tons, cooling_output_tons, it_heat_tons, margin_tons
```
