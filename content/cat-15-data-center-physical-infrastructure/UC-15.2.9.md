# UC-15.2.9 · Hot/Cold Aisle Temperature Delta

## Description

Low ΔT across containment indicates bypass airflow or insufficient airflow; high ΔT may signal restricted return.

## Value

Low ΔT across containment indicates bypass airflow or insufficient airflow; high ΔT may signal restricted return.

## Implementation

Pair sensors per row. Baseline expected ΔT for your containment design. Alert outside band.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Environmental sensors.
• Ensure the following data sources are available: `sourcetype="sensor:temperature"` position IN ("hot_aisle","cold_aisle").
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Pair sensors per row. Baseline expected ΔT for your containment design. Alert outside band.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=environment sourcetype="sensor:temperature" row_id=*
| bin _time span=5m
| stats avg(eval(if(position="cold_aisle",temp_c,null()))) as cold_c, avg(eval(if(position="hot_aisle",temp_c,null()))) as hot_c by row_id, _time
| eval delta_t=hot_c-cold_c
| where delta_t < 8 OR delta_t > 22
| table row_id, cold_c, hot_c, delta_t
```
