# UC-15.2.2 · Humidity Monitoring

## Description

Low humidity causes ESD risk; high humidity causes condensation. Maintaining 40-60% RH protects equipment.

## Value

Low humidity causes ESD risk; high humidity causes condensation. Maintaining 40-60% RH protects equipment.

## Implementation

Deploy humidity sensors alongside temperature sensors. Alert on out-of-range humidity (below 40% or above 60% RH). Track dew point to prevent condensation. Correlate with HVAC system humidifier/dehumidifier operation.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=environment sourcetype="sensor:humidity"
| where humidity_pct > 60 OR humidity_pct < 40
| table _time, zone, humidity_pct
```
