# UC-15.2.11 · Chiller Plant Efficiency (kW/ton)

## Description

Elevated kW/ton indicates fouling, low refrigerant, or poor tower performance.

## Value

Elevated kW/ton indicates fouling, low refrigerant, or poor tower performance.

## Implementation

Baseline design kW/ton from commissioning. Alert on sustained degradation. Schedule tube cleaning on trend.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: BMS chiller plant.
• Ensure the following data sources are available: `sourcetype="bms:chiller"` (tons, kw, cop).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Baseline design kW/ton from commissioning. Alert on sustained degradation. Schedule tube cleaning on trend.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cooling sourcetype="bms:chiller"
| eval kw_per_ton=round(kw/tons,3)
| where kw_per_ton > design_kw_per_ton * 1.15
| timechart span=15m avg(kw_per_ton) by chiller_id
```
