# UC-15.2.19 · Water Leak Sensor Zone Correlation

## Description

Correlates multiple rope sensors to localize leak source and trigger EPO workflows under raised floor.

## Value

Correlates multiple rope sensors to localize leak source and trigger EPO workflows under raised floor.

## Implementation

On any leak, show adjacent zones and nearest CRAC/isolation valve. Integrate with facilities runbook.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Leak detection panel → Splunk.
• Ensure the following data sources are available: `sourcetype="leak_detection"` (zone, sensor_id, conductivity).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
On any leak, show adjacent zones and nearest CRAC/isolation valve. Integrate with facilities runbook.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=environment sourcetype="leak_detection"
| where leak_detected=1
| stats count by zone, crac_id
| sort -count
```
