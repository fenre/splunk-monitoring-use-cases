# UC-15.2.13 · Air Handler Filter Differential Pressure

## Description

High ΔP across filters increases fan energy and reduces airflow; indicates change-out due.

## Value

High ΔP across filters increases fan energy and reduces airflow; indicates change-out due.

## Implementation

Set change-out threshold at ~85% of max rated ΔP. Correlate rising ΔP with fan speed increases.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: BMS AHU points.
• Ensure the following data sources are available: `sourcetype="bms:ahu"` (filter_dp_pa, fan_speed_pct).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Set change-out threshold at ~85% of max rated ΔP. Correlate rising ΔP with fan speed increases.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cooling sourcetype="bms:ahu"
| where filter_dp_pa > max_filter_dp_pa * 0.85
| table ahu_id, filter_dp_pa, max_filter_dp_pa, fan_speed_pct
```
