# UC-17.6.5 · Akamai Guardicore Network Flow Baseline Deviation Detection (`guardicore:network`)

## Description

Sudden tenfold jumps on unusual ports may be C2 or ransomware staging.

## Value

Detects threats without static IOC lists.

## Implementation

Install the add-on, configure API polling intervals appropriate to API limits, and map assets to CMDB for enrichment lookups.

## SPL

```spl
index=microseg sourcetype="guardicore:network" earliest=-7d
| bin _time span=1h
| stats count by dst_port, _time
| eventstats median(count) as med by dst_port
| eval z=(count-med)/med
| where z>3
```

## Visualization

Timechart (z-score), table.
