# UC-17.5.13 · Forcepoint Agent Health and Enrollment Coverage (`forcepoint:one`)

## Description

Agents off-patch or offline bypass web and DLP controls at the endpoint.

## Value

Endpoint teams hit coverage SLAs before audits.

## Implementation

Install the Insights SIEM app, configure HEC inputs per product line, and align indexes (`proxy`, `email`, `dlp`, `sase`). Normalize usernames to enterprise SAMAccountName.

## SPL

```spl
index=sase sourcetype="forcepoint:one" earliest=-4h
| where match(lower(event_type),"(?i)agent|endpoint")
| stats latest(status) as st latest(_time) as seen by hostname, agent_version
| where st!="healthy" OR seen < relative_time(now(),"-2h")
| sort seen
```

## Visualization

Single value (coverage %), table.
