# UC-17.5.14 · Forcepoint DLP Incident Escalation and Severity Progression (`forcepoint:dlp`)

## Description

Escalations indicate repeat offenders or growing data exposure.

## Value

Legal receives early warning when incidents trend toward breach thresholds.

## Implementation

Install the Insights SIEM app, configure HEC inputs per product line, and align indexes (`proxy`, `email`, `dlp`, `sase`). Normalize usernames to enterprise SAMAccountName.

## SPL

```spl
index=dlp sourcetype="forcepoint:dlp" earliest=-30d
| stats earliest(severity) as start_sev latest(severity) as end_sev by incident_id
| where start_sev!=end_sev
| sort incident_id
```

## Visualization

Table (incident, severity path), timeline.
