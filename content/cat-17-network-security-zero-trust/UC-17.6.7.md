# UC-17.6.7 · Akamai Guardicore Segmentation Policy Change Audit Trail (`guardicore:policy`)

## Description

Policy drift increases risk; auditing who relaxed rules supports least-privilege governance.

## Value

Forensics can tie breaches to a specific policy edit.

## Implementation

Install the add-on, configure API polling intervals appropriate to API limits, and map assets to CMDB for enrichment lookups.

## SPL

```spl
index=microseg sourcetype="guardicore:policy" earliest=-30d
| stats count values(change_type) as changes by admin_user, policy_name
| sort -count
```

## Visualization

Timeline, table.
