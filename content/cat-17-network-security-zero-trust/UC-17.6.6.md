# UC-17.6.6 · Akamai Guardicore Process-Level Communication Map Changes (`guardicore:network`)

## Description

New edges after code freeze windows often mean shadow IT or malware.

## Value

DevSecOps validates releases against observed dependencies.

## Implementation

Install the add-on, configure API polling intervals appropriate to API limits, and map assets to CMDB for enrichment lookups.

## SPL

```spl
index=microseg sourcetype="guardicore:network" earliest=-24h
| eval edge=src_asset."::".process_name."->".dst_asset."::".tostring(dst_port)
| stats earliest(_time) as first_seen by edge
| where first_seen > relative_time(now(),"-1d")
```
