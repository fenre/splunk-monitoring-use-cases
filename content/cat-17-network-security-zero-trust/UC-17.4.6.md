# UC-17.4.6 · Cloudflare Origin 5xx Error Rate Behind the Edge (`cloudflare:json`)

## Description

5xx from origin while Cloudflare is healthy means your application or hosting tier is failing—often invisible to internal monitoring if probes bypass the CDN.

## Value

Bridges the gap between CDN SRE and application owners with a single customer-impacting metric.

## Implementation

Configure Logpush jobs per dataset; route all into a unified `cloudflare:json` sourcetype or use transforms to normalize. Use `spath` for PascalCase fields. Create macros per zone name.

## SPL

```spl
index=cdn sourcetype="cloudflare:json" earliest=-24h
| spath OriginResponseStatus
| spath EdgeResponseStatus
| where OriginResponseStatus>=500
| timechart span=15m count by OriginResponseStatus
```

## Visualization

Timechart (origin 5xx), single value (SLI breach).
