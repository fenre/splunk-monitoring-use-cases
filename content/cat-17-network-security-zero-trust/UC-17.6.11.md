# UC-17.6.11 · Akamai Guardicore Agent Update Compliance and Version Drift (`guardicore:agent`)

## Description

Version skew hints at broken deployment pipelines or dormant VMs.

## Value

Security maintains a single supported agent baseline.

## Implementation

Install the add-on, configure API polling intervals appropriate to API limits, and map assets to CMDB for enrichment lookups.

## SPL

```spl
index=microseg sourcetype="guardicore:agent" earliest=-4h
| stats values(agent_version) as versions dc(hostname) as hosts by datacenter
| eval unique=mvsort(versions)
| where mvcount(unique)>1
| sort -hosts
```
