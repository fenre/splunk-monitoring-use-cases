# UC-17.1.8 · NAC Policy Change Audit

## Description

NAC policy changes affect network access for all devices. Unauthorized changes can create security gaps or disrupt access.

## Value

NAC policy changes affect network access for all devices. Unauthorized changes can create security gaps or disrupt access.

## Implementation

Forward ISE admin audit logs. Alert on any policy change. Track changes by administrator. Correlate with change management tickets. Report on policy change frequency.
