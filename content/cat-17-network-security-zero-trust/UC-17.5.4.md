# UC-17.5.4 · Forcepoint ONE CASB Shadow IT Application Discovery (`forcepoint:one`)

## Description

Unsanctioned SaaS spreads via credit-card signups; CASB logs reveal the real portfolio of cloud usage.

## Value

IT can broker sanctioned alternatives before data sprawls.

## Implementation

Install the Insights SIEM app, configure HEC inputs per product line, and align indexes (`proxy`, `email`, `dlp`, `sase`). Normalize usernames to enterprise SAMAccountName.

## SPL

```spl
index=sase sourcetype="forcepoint:one" earliest=-24h
| where match(lower(event_type),"(?i)casb|cloud app|saas")
| stats dc(app_name) as apps values(app_name) as examples by user
| where apps>12
| sort -apps
```

## Visualization

Table (user, app count), bar chart.
