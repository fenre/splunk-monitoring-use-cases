# UC-17.5.9 · Forcepoint Bandwidth Consumption by Application Category (`forcepoint:one`)

## Description

Understanding heavy categories guides QoS, caching, and SD-WAN policies.

## Value

Finance and infrastructure plan capacity with shared data.

## Implementation

Install the Insights SIEM app, configure HEC inputs per product line, and align indexes (`proxy`, `email`, `dlp`, `sase`). Normalize usernames to enterprise SAMAccountName.

## SPL

```spl
index=sase sourcetype="forcepoint:one" earliest=-24h
| stats sum(bytes_in) as in_b sum(bytes_out) as out_b by category
| eval total_mb=round((in_b+out_b)/1048576,2)
| sort -total_mb
```

## Visualization

Treemap (category bytes), table.
