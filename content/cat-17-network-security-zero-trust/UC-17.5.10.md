# UC-17.5.10 · Forcepoint Advanced Malware Detection File Analysis Outcomes (`forcepoint:one`)

## Description

AMD verdicts classify novel payloads; tracking outcomes measures advanced control efficacy.

## Value

Incident responders prioritize hosts downloading malicious verdict files.

## Implementation

Install the Insights SIEM app, configure HEC inputs per product line, and align indexes (`proxy`, `email`, `dlp`, `sase`). Normalize usernames to enterprise SAMAccountName.

## SPL

```spl
index=sase sourcetype="forcepoint:one" earliest=-7d
| where match(lower(event_type),"(?i)malware|sandbox|amd")
| stats count by verdict, file_type, sha256
| sort -count
```

## Visualization

Pie (verdict), table (hash).
