# UC-17.5.12 · Forcepoint Administrator Configuration Change Audit (`forcepoint:one`)

## Description

SSE consoles are high-value targets; immutable logs of admin edits support insider investigations.

## Value

Change boards correlate outages with policy publishes.

## Implementation

Install the Insights SIEM app, configure HEC inputs per product line, and align indexes (`proxy`, `email`, `dlp`, `sase`). Normalize usernames to enterprise SAMAccountName.

## SPL

```spl
index=sase sourcetype="forcepoint:one" earliest=-24h
| where match(lower(event_type),"(?i)admin|config|policy.publish")
| stats count by admin_user, object, action
| sort -count
```
