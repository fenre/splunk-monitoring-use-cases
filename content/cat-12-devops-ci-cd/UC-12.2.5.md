# UC-12.2.5 · Failed Deployment Tracking

## Description

Failed deployments cause service disruption. Rapid detection enables rollback decisions. Change failure rate is a DORA metric.

## Value

Failed deployments cause service disruption. Rapid detection enables rollback decisions. Change failure rate is a DORA metric.

## Implementation

Track all deployment outcomes including failures and rollbacks. Alert immediately on production deployment failures. Calculate change failure rate (DORA metric). Correlate with application error rate to measure deployment impact.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl

| table _time, application, environment, version, deployer, error_message
| sort -_time
```
