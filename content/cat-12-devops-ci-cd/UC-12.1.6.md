# UC-12.1.6 · Force Push Detection

## Description

Force pushes overwrite git history, potentially destroying code and audit trails. Detection ensures accountability.

## Value

Force pushes overwrite git history, potentially destroying code and audit trails. Detection ensures accountability.

## Implementation

Parse force push flag from webhook events. Alert on any force push to shared branches. Whitelist expected force pushes (e.g., squash-merge workflows on feature branches). Track frequency per developer.

## Detailed Implementation
