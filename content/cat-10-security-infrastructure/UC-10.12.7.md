# UC-10.12.7 · PCI DSS Log Review Compliance

## Description

Demonstrates ongoing review of security events for cardholder data environments (CDE) per PCI DSS Req. 10–12.

## Value

Demonstrates ongoing review of security events for cardholder data environments (CDE) per PCI DSS Req. 10–12.

## Implementation

Tag CDE assets; weekly attested review searches with saved reports and sign-off in GRC tooling.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Enterprise Security, file integrity / FIM TA.
• Ensure the following data sources are available: `sourcetype="linux:audit"`, firewall, IDS in `tag=pci`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Tag CDE assets; weekly attested review searches with saved reports and sign-off in GRC tooling.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=pci (sourcetype="linux:audit" OR sourcetype="pan:traffic") tag=pci
| stats count by sourcetype, host, severity
| join type=left max=1 host [| inputlookup pci_asset_inventory.csv | fields host reviewer last_review]
| where isnull(last_review) OR last_review < relative_time(now(),"-7d@d")
```

**Pipeline walkthrough**

• Scopes the data: index=pci, sourcetype="linux:audit", tags. Cross-check against **Data sources** above so indexes and sourcetypes match your ingestion.
• `stats` rolls up events into metrics; results are split **by sourcetype, host, severity** so each row reflects one combination of those dimensions (useful for per-host, per-user, or per-entity comparisons for this use case).
• Joins to a subsearch with `join` — set `max=` to match cardinality and avoid silent truncation.
• Filters the current rows with `where isnull(last_review) OR last_review < relative_time(now(),"-7d@d")` — typically the threshold or rule expression for this monitoring goal.
