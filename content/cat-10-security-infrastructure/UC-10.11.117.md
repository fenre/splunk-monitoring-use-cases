# UC-10.11.117 · Tanium Interact Saved Question Drift

## Description

Saved question drift avoids acting on outdated inventory logic.

## Value

Saved question drift avoids acting on outdated inventory logic.

## Implementation

Find saved questions not updated in 180 days. Retire or refresh for accuracy.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_tanium`.
• Ensure the following data sources are available: `sourcetype=tanium:saved`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Find saved questions not updated in 180 days. Retire or refresh for accuracy.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=endpoint sourcetype="tanium:saved"
| stats latest(mod_time) as m by question_name
| where now()-m > 15552000
```
