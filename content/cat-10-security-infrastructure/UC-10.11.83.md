# UC-10.11.83 · Tanium Real-Time Question Response Times

## Description

Response times matter when Tanium is used for incident containment questions.

## Value

Response times matter when Tanium is used for incident containment questions.

## Implementation

Monitor Interact latency for live investigations. Alert when platform or network degrades.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_tanium`.
• Ensure the following data sources are available: `sourcetype=tanium:interact`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Monitor Interact latency for live investigations. Alert when platform or network degrades.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=endpoint sourcetype="tanium:interact"
| stats perc95(response_ms) as p95 by question_name
| where p95>5000
```
