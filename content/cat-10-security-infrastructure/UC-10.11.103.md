# UC-10.11.103 · CrowdStrike Cloud Workload Protection Detections

## Description

CWP extends Falcon detection to cloud-native workloads.

## Value

CWP extends Falcon detection to cloud-native workloads.

## Implementation

Ingest CWP for containers and VMs. Map to Kubernetes namespace ownership.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `TA-crowdstrike-falcon`.
• Ensure the following data sources are available: `sourcetype=crowdstrike:cwp`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Ingest CWP for containers and VMs. Map to Kubernetes namespace ownership.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cloud sourcetype="crowdstrike:cwp"
| stats count by cluster, image, technique
| sort -count
```
