# UC-10.13.7 · Off-Hours Authentication (CIM)

## Description

Privileged or bulk success auths outside business hours warrant review.

## Value

Privileged or bulk success auths outside business hours warrant review.

## Implementation

Join HR schedule lookup; reduce noise for global teams.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: CIM Authentication.
• Ensure the following data sources are available: Normalized `_time` with user timezone optional.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Join HR schedule lookup; reduce noise for global teams.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=authentication action=success
| eval hr=strftime(_time,"%H")
| where (hr<6 OR hr>22) AND (match(user,"^admin") OR match(user,"^svc_"))
| stats count by user, app, src
```
