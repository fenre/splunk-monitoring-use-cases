# UC-10.11.105 · CrowdStrike Custom IOA Tuning Metrics

## Description

IOA tuning keeps behavioral detection accurate for your environment.

## Value

IOA tuning keeps behavioral detection accurate for your environment.

## Implementation

Measure false positive rate per custom IOA. Retire or refine noisy rules.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `TA-crowdstrike-falcon`.
• Ensure the following data sources are available: `sourcetype=crowdstrike:ioa`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Measure false positive rate per custom IOA. Retire or refine noisy rules.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=endpoint sourcetype="crowdstrike:ioa"
| stats count by rule_id, disposition
| where disposition="false_positive"
| sort -count
```
