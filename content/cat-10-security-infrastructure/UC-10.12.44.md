# UC-10.12.44 · CJIS Audit Log Compliance

## Description

Ensures CJIS systems capture user ID, timestamp, device, and action for NCIC/III access as required.

## Value

Ensures CJIS systems capture user ID, timestamp, device, and action for NCIC/III access as required.

## Implementation

Monthly completeness report; tamper-evident forwarding; background check status on accounts.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: RMS/CAD, CJIS WAN.
• Ensure the following data sources are available: `sourcetype="cjis:query"`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Monthly completeness report; tamper-evident forwarding; background check status on accounts.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cjis sourcetype="cjis:query"
| where isnull(officer_id) OR isnull(ncic_transaction_id) OR len(device_id)<4
| stats count by agency, src, query_type
| sort -count
```
