# UC-10.11.109 · Zscaler Bandwidth Control and QoS Events

## Description

QoS events explain user complaints and detect bandwidth abuse.

## Value

QoS events explain user complaints and detect bandwidth abuse.

## Implementation

Review throttle events for fairness and abuse. Distinguish policy from congestion.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_zscaler`.
• Ensure the following data sources are available: `sourcetype=zscaler:qos`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 2 — Create the search and alert
