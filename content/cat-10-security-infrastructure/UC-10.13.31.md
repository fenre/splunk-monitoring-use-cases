# UC-10.13.31 · Attachment Type Analysis

## Description

Tracks `file_name` extensions / MIME for macro and archive risk.

## Value

Tracks `file_name` extensions / MIME for macro and archive risk.

## Implementation

Alert on `.iso`, `.lnk` spikes from external senders.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Email CIM.
• Ensure the following data sources are available: `file_name`, `signature`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Alert on `.iso`, `.lnk` spikes from external senders.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=email
| rex field=file_name "(?<ext>\.[a-zA-Z0-9]+)$"
| stats count by ext
| sort -count
```
