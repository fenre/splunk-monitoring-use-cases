Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_paloalto`.

Step 1 — Configure data collection
Filter threat subtype spyware. Correlate with DNS and URL logs. Tune signatures generating noise.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=pan sourcetype="pan:threat" subtype="spyware"
| timechart span=1h count by threat_id
| where count>0
```
