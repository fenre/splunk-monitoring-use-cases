# UC-10.11.56 · ZIA SSL Inspection Coverage

## Description

SSL coverage ensures encrypted threats are visible where policy allows inspection.

## Value

SSL coverage ensures encrypted threats are visible where policy allows inspection.

## Implementation

Calculate inspected versus bypassed SSL sessions. Investigate certificate pinning or legal bypass lists.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_zscaler`.
• Ensure the following data sources are available: `sourcetype=zscaler:ssl`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Calculate inspected versus bypassed SSL sessions. Investigate certificate pinning or legal bypass lists.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=proxy sourcetype="zscaler:ssl"
| stats count as total, sum(eval(if(inspected==1,1,0))) as insp
| eval pct=round(100*insp/total,2)
| where pct<85
```
