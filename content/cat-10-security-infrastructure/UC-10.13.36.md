# UC-10.13.36 · User Agent Analysis

## Description

Rare or known-bad `http_user_agent` strings — scraping and legacy clients.

## Value

Rare or known-bad `http_user_agent` strings — scraping and legacy clients.

## Implementation

Lookup `ua_badlist`; alert on empty or SQLi-like UAs.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Web CIM.
• Ensure the following data sources are available: `http_user_agent`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Lookup `ua_badlist`; alert on empty or SQLi-like UAs.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=web
| stats count by http_user_agent
| sort 100 -count
```
