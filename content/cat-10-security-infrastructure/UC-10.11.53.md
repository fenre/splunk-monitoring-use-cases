# UC-10.11.53 · Zscaler Cloud Application Discovery

## Description

Cloud app discovery informs CASB and firewall policy for sanctioned tools.

## Value

Cloud app discovery informs CASB and firewall policy for sanctioned tools.

## Implementation

Use cloud app visibility feeds. Flag unsanctioned SaaS with high upload volume.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_zscaler`.
• Ensure the following data sources are available: `sourcetype=zscaler:cloudapp`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Use cloud app visibility feeds. Flag unsanctioned SaaS with high upload volume.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cloud sourcetype="zscaler:cloudapp"
| stats dc(app_name) as apps by department
| sort -apps
```
