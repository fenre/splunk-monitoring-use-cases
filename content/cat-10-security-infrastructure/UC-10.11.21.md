# UC-10.11.21 · Palo Alto Zone-Based Firewall Violations

## Description

Zone violations reveal lateral movement attempts and policy gaps between segments.

## Value

Zone violations reveal lateral movement attempts and policy gaps between segments.

## Implementation

Use traffic logs with zone fields. Baseline expected east-west patterns. Alert on denies involving sensitive zones.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_paloalto`.
• Ensure the following data sources are available: `sourcetype=pan:traffic`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Use traffic logs with zone fields. Baseline expected east-west patterns. Alert on denies involving sensitive zones.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=pan sourcetype="pan:traffic" action=deny
| stats count by src_zone, dst_zone, rule
| sort -count
```
