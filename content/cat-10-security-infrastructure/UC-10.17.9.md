# UC-10.17.9 · SonicWall Application Control Policy Violations (SonicWall)

## Description

Application awareness stops unsanctioned remote-access tools and crypto miners. Violation trending shows policy effectiveness.

## Value

Supports zero-trust application allow-listing initiatives with measurable metrics.

## Implementation

Map SonicWall application IDs to names. Exclude approved IT tools via lookup.

## SPL

```spl
index=netfw sourcetype="dell:sonicwall:firewall" earliest=-7d
| where match(_raw,"(?i)AppCtrl|application control")
| stats count by app_name, usr, fw_action
| sort -count
```

## Visualization

Bar chart (application), table (user, app, action).
