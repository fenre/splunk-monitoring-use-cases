# UC-10.13.26 · Remediation Velocity

## Description

Median time from `first_found` to `status=closed` by team — CIM-normalized across scanners.

## Value

Median time from `first_found` to `status=closed` by team — CIM-normalized across scanners.

## Implementation

Requires closed timestamp in data; else use ticket linkage.

• Install and configure the required add-on or app: Vulnerabilities CIM.

• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Requires closed timestamp in data; else use ticket linkage.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=vulnerability status=closed
| eval mttr=(closed_time-first_found_epoch)/3600
| stats median(mttr) as med_hrs by owner_team
```
