# UC-10.11.111 · VMware Carbon Black Vulnerability Assessment Integration

## Description

Integrated vuln view reduces tool sprawl between EDR and VM teams.

## Value

Integrated vuln view reduces tool sprawl between EDR and VM teams.

## Implementation

Join CB vuln data with patch tickets. Prioritize criticals on internet-facing hosts.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_vmware_carbonblack`.
• Ensure the following data sources are available: `sourcetype=carbonblack:vuln`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Join CB vuln data with patch tickets. Prioritize criticals on internet-facing hosts.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=endpoint sourcetype="carbonblack:vuln"
| stats count by cve_id, severity
| where patch_status="missing" AND severity IN ("CRITICAL","HIGH")
| sort -severity
```
