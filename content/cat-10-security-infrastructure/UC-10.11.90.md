# UC-10.11.90 · FortiClient EMS Compliance Posture

## Description

FortiClient posture extends firewall policy to the endpoint before network admission.

## Value

FortiClient posture extends firewall policy to the endpoint before network admission.

## Implementation

Ingest FortiClient telemetry for AV, patch, and fabric compliance. Block non-compliant VPN.
