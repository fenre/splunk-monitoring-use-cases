# UC-10.13.18 · Malware Family Distribution

## Description

Shifts toward ransomware or infostealer families prioritize IR playbooks.

## Value

Shifts toward ransomware or infostealer families prioritize IR playbooks.

## Implementation

Maintain `malware_family_lookup` for vendor-specific strings.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Malware CIM.
• Ensure the following data sources are available: `signature` or `user` field for family tags.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Maintain `malware_family_lookup` for vendor-specific strings.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=malware
| rex field=signature "(?<family>[A-Za-z0-9]+)"
| stats count by family
| sort -count
```
