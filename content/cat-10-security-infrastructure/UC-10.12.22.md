# UC-10.12.22 · Infusion Pump Connectivity Monitoring

## Description

Pump disconnects can delay therapy; supports Joint Commission device management expectations.

## Value

Pump disconnects can delay therapy; supports Joint Commission device management expectations.

## Implementation

Integrate with CMMS for maintenance; alert biomed on extended silence from critical care beds.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Medical device integration (MDI), RTLS.
• Ensure the following data sources are available: `sourcetype="mdi:pump"`, `sourcetype="hl7:mdt"`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Integrate with CMMS for maintenance; alert biomed on extended silence from critical care beds.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=clinical sourcetype="mdi:pump"
| stats latest(_time) as last_seen, latest(alarm_state) as alarm by pump_id, unit
| eval gap_sec=now()-last_seen
| where gap_sec>300 OR alarm="AIR_IN_LINE"
| table pump_id, unit, gap_sec, alarm
```
