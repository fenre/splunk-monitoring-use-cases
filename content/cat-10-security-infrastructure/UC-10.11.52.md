# UC-10.11.52 · ZIA DLP Incident Analysis

## Description

ZIA DLP analysis reduces data exfiltration via web and SaaS paths.

## Value

ZIA DLP analysis reduces data exfiltration via web and SaaS paths.

## Implementation

Collect DLP events from web and cloud channels. Classify by data identifier severity.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_zscaler`.
• Ensure the following data sources are available: `sourcetype=zscaler:dlp`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Collect DLP events from web and cloud channels. Classify by data identifier severity.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=dlp sourcetype="zscaler:dlp" dlp_action=block
| stats count by policy, channel, user
| sort -count
```
