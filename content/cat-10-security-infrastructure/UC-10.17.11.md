# UC-10.17.11 · SonicWall Per-Zone Interface Bandwidth Utilization (SonicWall)

## Description

Understanding bytes per zone prevents saturated uplinks from masquerading as application bugs.

## Value

Guides upgrades and QoS changes with data instead of anecdotes.

## Implementation

If only syslog throughput is available, supplement with SNMP polling scripted input for `rx_bps`/`tx_bps`.

## SPL

```spl
index=netops sourcetype="dell:sonicwall" earliest=-24h
| where match(_raw,"(?i)interface|X1|X2|zone")
| rex field=_raw "(?i)(?<iface>X\d+|ae\d+).*?(?<kbps>\d+)\s*kbps"
| timechart span=15m max(kbps) by iface
```

## Visualization

Timechart (bps by zone), area chart (95th percentile).
