# UC-10.11.55 · ZDX User Experience Scores

## Description

ZDX scores tie security service performance to user productivity and help desk tickets.

## Value

ZDX scores tie security service performance to user productivity and help desk tickets.

## Implementation

Onboard ZDX synthetic and real user metrics. Correlate poor scores with ISP or app outages.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_zscaler`.
• Ensure the following data sources are available: `sourcetype=zscaler:zdx`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Onboard ZDX synthetic and real user metrics. Correlate poor scores with ISP or app outages.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ux sourcetype="zscaler:zdx"
| stats avg(score) as avg_score by user, app
| where avg_score<70
```
