# UC-10.11.43 · CrowdStrike Falcon Discover Asset Inventory Drift

## Description

Asset drift finds shadow systems missing EDR or patching.

## Value

Asset drift finds shadow systems missing EDR or patching.

## Implementation

Ingest Discover unmanaged assets. Compare to CMDB. Alert on stale or unknown devices.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `TA-crowdstrike-falcon`.
• Ensure the following data sources are available: `sourcetype=crowdstrike:discover`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Ingest Discover unmanaged assets. Compare to CMDB. Alert on stale or unknown devices.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=endpoint sourcetype="crowdstrike:discover"
| stats latest(last_seen) as ls by hostname, aid
| eval days_since=(now()-ls)/86400
| where days_since>30
```
