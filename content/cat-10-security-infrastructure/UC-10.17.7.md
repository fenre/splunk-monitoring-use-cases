# UC-10.17.7 · SonicWall SSL DPI Bypass and Decryption Policy Exceptions (SonicWall)

## Description

Decrypting TLS is contentious but often required for compliance. Logging bypasses ensures exceptions are rare, approved, and time-bounded.

## Value

Auditors receive proof that sensitive traffic is either inspected or explicitly exempted with justification.

## Implementation

Tag bypass events with change ticket IDs via lookup on rule name. Alert on new bypass destinations.

## SPL

```spl
index=netfw sourcetype="dell:sonicwall:firewall" earliest=-7d
| where match(_raw,"(?i)SSL DPI|decrypt|inspection.*bypass|exclude")
| stats count by rule, dst_ip, usr
| sort -count
```
