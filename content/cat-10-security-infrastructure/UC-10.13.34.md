# UC-10.13.34 · HTTP Method Distribution

## Description

Surge in `PUT`/`DELETE` or rare methods may indicate API abuse.

## Value

Surge in `PUT`/`DELETE` or rare methods may indicate API abuse.

## Implementation

Per-`url` allowlists for admin APIs.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: WAF, proxy → Web.
• Ensure the following data sources are available: `http_method` in `Web` model.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Per-`url` allowlists for admin APIs.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=web
| stats count by http_method
| sort -count
```
