# UC-10.11.69 · Carbon Black Network Isolation Events

## Description

## Value

Isolation events contain active threats while preserving forensic options.

## Implementation

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: `Splunk_TA_vmware_carbonblack`.
• Ensure the following data sources are available: `sourcetype=carbonblack:isolation`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Track isolate and release events with analyst or automated source. Document for IR.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=endpoint sourcetype="carbonblack:isolation"
| stats count by action, reason, hostname
| sort -_time
```
