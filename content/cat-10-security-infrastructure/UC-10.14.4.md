# UC-10.14.4 · PLC Program Change Outside Maintenance Window

## Description

Flags logic downloads or program changes to PLCs outside approved windows (T0845), protecting process integrity and safety interlocks.

## Value

Flags logic downloads or program changes to PLCs outside approved windows (T0845), protecting process integrity and safety interlocks.

## Implementation

Requires Splunk ES and OT Security Add-on tied to CMMS or change tickets. Validate engineering station identity. Feed ES notables for T0845 when no matching maintenance record exists.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: OT Security Add-on for Splunk (Splunkbase 5151).
• Ensure the following data sources are available: `index=ot` `sourcetype="ics:plc"` or engineering tool audit logs.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Requires Splunk ES and OT Security Add-on tied to CMMS or change tickets. Validate engineering station identity. Feed ES notables for T0845 when no matching maintenance record exists.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ot sourcetype="ics:plc"
| where match(action, "(?i)download|upload|program|transfer")
| lookup ot_maintenance_windows plc_id OUTPUT window_start window_end
| eval in_window=if(_time>=window_start AND _time<=window_end, 1, 0)
| where in_window=0 OR isnull(window_start)
| stats count values(action) as actions by plc_id, src_host, user
```
