# UC-10.13.32 · Internal-External Mail Ratio

## Description

Sudden flip to mostly external recipients may indicate data exfil via email or open relay abuse.

## Value

Sudden flip to mostly external recipients may indicate data exfil via email or open relay abuse.

## Implementation

Baseline per department; alert when `ext_ratio` >2× normal.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Email CIM.
• Ensure the following data sources are available: `recipient`, `src_user` with domain classification.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Baseline per department; alert when `ext_ratio` >2× normal.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=email
| eval ext=if(match(recipient,"@ourdomain\.com$"),0,1)
| timechart span=1h sum(ext) as ext_count, count as tot
| eval ext_ratio=ext_count/tot
```
