# UC-10.13.44 · Process Execution Anomaly

## Description

Rare `process_name` on endpoint fleet — living-off-the-land and malware binaries.

## Value

Rare `process_name` on endpoint fleet — living-off-the-land and malware binaries.

## Implementation

Focus on `process_path` under user-writable dirs; elevate if `parent_process_name` is office apps.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Sysmon, EDR → Endpoint.Processes.
• Ensure the following data sources are available: `Processes` in Endpoint model.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Focus on `process_path` under user-writable dirs; elevate if `parent_process_name` is office apps.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=security tag=endpoint tag=process
| stats count by process_name, host
| eventstats median(count) as med by process_name
| where count < med*0.01 AND count>0
```
