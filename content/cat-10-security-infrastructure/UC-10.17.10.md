# UC-10.17.10 · SonicWall WAN Failover and Load-Balance Path Events (SonicWall)

## Description

WAN transitions directly affect SaaS and voice quality. Structured monitoring reduces MTTR when a circuit or probe fails.

## Value

Operations teams validate SLA credits and prove redundant path testing.

## Implementation

Enable SNMP traps or syslog for HA/LLB events. Correlate with carrier tickets.

## SPL

```spl
index=netops sourcetype="dell:sonicwall" earliest=-24h
| where match(_raw,"(?i)failover|LB group|probe.*fail|link.*down|WAN")
| stats count by host, message
| sort -count
```

## Visualization

Timeline (failover markers), single value (active path).
