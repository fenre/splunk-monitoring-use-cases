# UC-21.3.3 · Nurse Call System Response Time

## Description

Nurse call response time is a key patient satisfaction and safety metric. Monitoring enables staffing optimization and regulatory compliance.

## Value

Nurse call response time is a key patient satisfaction and safety metric. Monitoring enables staffing optimization and regulatory compliance.

## Implementation

Integrate nurse call system events via syslog or API. Track response times by unit and shift. Alert when p95 exceeds 5 minutes. Report for CMS quality metrics. **Domain context:** Response time is a common patient experience and safety proxy; thresholds vary by acuity level (e.g. ICU vs med-surg)—segment by `unit` or priority if available. **Splunk:** Confirm whether `response_time` is seconds or milliseconds; normalize before `stats`.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=healthcare sourcetype="nursecall:event"
| eval response_sec=response_time
| bin _time span=1h
| stats avg(response_sec) as avg_response, perc95(response_sec) as p95_response, count by unit, _time
| where p95_response > 300
| table _time, unit, avg_response, p95_response, count
```
