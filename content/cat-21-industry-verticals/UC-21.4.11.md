# UC-21.4.11 · Intermodal Container Dwell Time

## Description

Excessive container dwell time at terminals increases demurrage costs and reduces asset turns. Tracking dwell enables process improvements and better planning.

## Value

Excessive container dwell time at terminals increases demurrage costs and reduces asset turns. Tracking dwell enables process improvements and better planning.

## Implementation

Define dwell from ingate to outgate. Compare rail vs truck facilities. Target reduction projects at high-p95 sites. **Domain context:** Demurrage/detention charges incentivize dwell reduction; intermodal metrics often feed into shipper scorecards (OTIF). **Splunk:** If `dwell_hours` is snapshot, not event-derived, confirm refresh cadence to avoid stale rankings.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=logistics sourcetype="container:dwell"
| where isnotnull(dwell_hours)
| stats avg(dwell_hours) as avg_dwell, perc95(dwell_hours) as p95_dwell, max(dwell_hours) as max_dwell by facility
| sort - p95_dwell
| table facility, avg_dwell, p95_dwell, max_dwell
```
