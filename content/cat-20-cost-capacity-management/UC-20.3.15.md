# UC-20.3.15 · GitHub Enterprise Seat Utilization versus Active Contributors

## Description

GitHub Enterprise bills per occupied seat while many accounts represent bots, service users, or former contractors. Comparing billed seats to users with pushes or reviews in the last sixty days supports deprovisioning and org consolidation before annual renewal.

## Value

GitHub Enterprise bills per occupied seat while many accounts represent bots, service users, or former contractors. Comparing billed seats to users with pushes or reviews in the last sixty days supports deprovisioning and org consolidation before annual renewal.

## Implementation

(1) Enable audit log streaming with actor and action fields normalized. (2) Ingest nightly seat count from billing API into `github:license_snapshot`. (3) Coordinate dormant seat removal with org owners outside peak release windows.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: GitHub Audit Log streaming to Splunk, GitHub Enterprise Server TA (optional).
• Ensure the following data sources are available: `index=devops` `sourcetype="github:audit"`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Enable audit log streaming with actor and action fields normalized. (2) Ingest nightly seat count from billing API into `github:license_snapshot`. (3) Coordinate dormant seat removal with org owners outside peak release windows.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=devops sourcetype="github:audit" earliest=-60d
| search action=pull_request OR action=push OR action=issue_comment
| eval actor=coalesce(actor, user, login)
| stats dc(actor) as active_contributors_60d by enterprise_slug
| join type=left enterprise_slug [
  search index=devops sourcetype="github:license_snapshot" earliest=-1d
  | stats latest(billed_seats) as billed_seats by enterprise_slug
]
| eval seat_util_pct=round(100*active_contributors_60d/nullif(billed_seats,0),1)
| eval dormant_seats=billed_seats-active_contributors_60d
| where dormant_seats>10 OR seat_util_pct<70
| table enterprise_slug, billed_seats, active_contributors_60d, seat_util_pct, dormant_seats
```
