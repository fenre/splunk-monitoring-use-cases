# UC-20.3.11 · Oracle Database Option Usage versus Entitlements

## Description

Options such as Partitioning, Advanced Compression, and Diagnostics Pack are metered separately from the processor license. Undetected feature use creates audit exposure and six-figure true-up risk; proactive comparison to LMS entitlements funds remediation projects instead of penalties.

## Value

Options such as Partitioning, Advanced Compression, and Diagnostics Pack are metered separately from the processor license. Undetected feature use creates audit exposure and six-figure true-up risk; proactive comparison to LMS entitlements funds remediation projects instead of penalties.

## Implementation

(1) Ingest weekly DBA_FEATURE_USAGE_STATISTICS via DB Connect with stable `product` names. (2) Maintain `oracle:entitlement` from procurement. (3) Open high-priority changes to disable unused options or purchase entitlements before vendor review.

## Detailed Implementation

Prerequisites

• Ensure the following data sources are available: `index=database` `sourcetype="oracle:option_usage"`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Ingest weekly DBA_FEATURE_USAGE_STATISTICS via DB Connect with stable `product` names. (2) Maintain `oracle:entitlement` from procurement. (3) Open high-priority changes to disable unused options or purchase entitlements before vendor review.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=database sourcetype="oracle:option_usage" earliest=-1d
| eval detected=if(upper(currently_used)="TRUE" OR currently_used="1",1,0)
| stats values(product) as option_name max(detected) as in_use by db_name, host
| join type=left db_name host [
  search index=licenses sourcetype="oracle:entitlement" earliest=-1d
  | table db_name, host, product, entitled
]
| eval gap=if(in_use=1 AND (entitled=0 OR isnull(entitled)),1,0)
| where gap=1
| table db_name, host, option_name, entitled
```
