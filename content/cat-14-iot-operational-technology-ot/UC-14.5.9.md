# UC-14.5.9 · OPC-UA Certificate Expiration and Trust

## Description

Expired or untrusted OPC-UA certificates break secure connections and prevent data collection. Proactive monitoring avoids blind spots after certificate rollover failures.

## Value

Expired or untrusted OPC-UA certificates break secure connections and prevent data collection. Proactive monitoring avoids blind spots after certificate rollover failures.

## Implementation

Forward OPC-UA connector and gateway logs. Parse certificate and trust-related messages. Maintain a script or lookup of cert expiry dates; alert when expiry is within 30 days or when log shows validation failure.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Edge Hub OPC-UA connector, OPC-UA gateway.
• Ensure the following data sources are available: Connector/gateway logs (certificate validation, trust list).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md
