# UC-14.1.37 · Elevator Door Fault Frequency and Prediction

## Description

Door faults are the most common elevator malfunction and the leading cause of entrapments. Monitoring door close force, cycle time, and motor amperage detects deteriorating door systems 2-6 weeks before failure — preventing passenger entrapments and service calls.

## Value

Door faults are the most common elevator malfunction and the leading cause of entrapments. Monitoring door close force, cycle time, and motor amperage detects deteriorating door systems 2-6 weeks before failure — preventing passenger entrapments and service calls.

## Implementation

Collect door cycle events including close force, cycle time, and reversal events. Trend door close time — increasing values indicate worn rollers, dirty tracks, or actuator degradation. Alert when fault rate exceeds 1% or close time exceeds 5 seconds.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Elevator controller API, vibration/current sensors via IoT gateway.
• Ensure the following data sources are available: `sourcetype=bms:elevator`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Collect door cycle events including close force, cycle time, and reversal events. Trend door close time — increasing values indicate worn rollers, dirty tracks, or actuator degradation. Alert when fault rate exceeds 1% or close time exceeds 5 seconds.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=building sourcetype="bms:elevator" event_type="door_*"
| eval is_fault=if(event_type="door_fault" OR door_close_time_ms > 5000 OR door_reversal=1, 1, 0)
| bin _time span=1d
| stats sum(is_fault) as daily_faults, count as total_door_cycles, avg(door_close_time_ms) as avg_close_time, sum(door_reversal) as reversals by elevator_id, _time
| eval fault_rate_pct=round(daily_faults*100/total_door_cycles, 2)
| where fault_rate_pct > 1 OR daily_faults > 5
| table _time, elevator_id, daily_faults, total_door_cycles, fault_rate_pct, avg_close_time, reversals
```
