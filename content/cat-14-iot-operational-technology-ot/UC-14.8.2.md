# UC-14.8.2 · Sensor Data Quality Trending

## Description

Missing or stale readings break historians, dashboards, and ML features; daily trending exposes ingestion gaps early so teams can fix collectors, radios, or brokers before safety or quality thresholds are blind.

## Value

Missing or stale readings break historians, dashboards, and ML features; daily trending exposes ingestion gaps early so teams can fix collectors, radios, or brokers before safety or quality thresholds are blind.

## Implementation

Map your quality field (`good`, `192` OPC status, Modbus exception flags). If quality is not present, infer staleness with `streamstats` per `device_id` and `tag` using gaps between `_time` greater than `3×` the expected poll interval from a lookup. Dedupe by `device_id`+`tag` to avoid double counting. Tune thresholds per line speed—fast PLCs need tighter windows than environmental sensors.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Edge Hub, Modbus/OPC-UA TAs.
• Ensure the following data sources are available: `index=ot` `sourcetype IN ("edge_hub:metrics","edge_hub:events","modbus:readings","opcua:telemetry")`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
Map your quality field (`good`, `192` OPC status, Modbus exception flags). If quality is not present, infer staleness with `streamstats` per `device_id` and `tag` using gaps between `_time` greater than `3×` the expected poll interval from a lookup. Dedupe by `device_id`+`tag` to avoid double counting. Tune thresholds per line speed—fast PLCs need tighter windows than environmental sensors.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ot sourcetype IN ("edge_hub:metrics","modbus:readings","opcua:telemetry") earliest=-30d@d
| eval bad=if(match(lower(coalesce(quality,data_quality)),"(?i)bad|invalid|stale") OR isnull(metric_value),1,0)
| timechart span=1d sum(bad) as bad_reads count as total_reads
| eval bad_pct=round(100*bad_reads/nullif(total_reads,0),2)
| trendline sma7(bad_pct) as quality_trend
| eventstats avg(bad_pct) as run_avg
| eval delta=round(bad_pct - run_avg,2)
```
