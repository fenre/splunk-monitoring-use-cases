# 22.28.15 — Grundschutz certification evidence pack

**Criticality:** critical | **Difficulty:** intermediate | **Status:** N/A

## Description

Demonstrates alignment with BSI IT-Grundschutz expectations for grundschutz certification evidence pack using auditable Splunk evidence.

## Value

Gives the responsible team auditor-ready evidence for IT-Grundschutz compliance by automating regulatory compliance evidence using indexes audit and sourcetypes grundschutz:cert. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=audit sourcetype="grundschutz:cert" earliest=-365d
| stats latest(expiry) as ex, latest(level) as lvl by scope_id
| eval ex_epoch=strptime(ex,"%Y-%m-%d")
| where ex_epoch<relative_time(now(),"+90d@d")
| table scope_id, lvl, ex
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Vulnerabilities.Vulnerabilities by Vulnerabilities.severity, Vulnerabilities.signature, Vulnerabilities.dest | sort - count
```
