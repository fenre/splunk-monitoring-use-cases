# UC-22.9.8 · Compliance Trending — Auditor Evidence Pack Generation Volume and Deficiency Rate

## Description

External audits consume evidence packs. Trending how many packs were produced, how often they were rejected or re-requested for gaps, and median assembly time shows whether the organization is maturing its evidence factory—not scrambling at year-end.

## Value

Gives the responsible team auditor-ready evidence for SOC 2, SOX ITGC compliance by automating regulatory compliance evidence using indexes grc and sourcetypes audit:evidence_pack. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Define `deficiency_flag` from auditor workflow state; (2) redact `reviewer_comment_len` only—never store full comments with PII; (3) integrate with secure file transfer metrics optionally; (4) target downward trend on `deficiency_rate`; (5) map `engagement_id` to regulator vs commercial audit classes. Indexes required: `index=grc`; Sourcetypes: `sourcetype=audit:evidence_pack`.

• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Define `deficiency_flag` from auditor workflow state; (2) redact `reviewer_comment_len` only—never store full comments with PII; (3) integrate with secure file transfer metrics optionally; (4) target downward trend on `deficiency_rate`; (5) map `engagement_id` to regulator vs commercial audit classes.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=grc sourcetype="audit:evidence_pack" earliest=-730d
| eval created=strptime(created_ts,"%Y-%m-%dT%H:%M:%SZ"), sent=strptime(sent_ts,"%Y-%m-%dT%H:%M:%SZ")
| eval assemble_h=round((sent-created)/3600,1)
| bin sent span=90d as quarter
| stats count as packs, sum(deficiency_flag) as deficient, median(assemble_h) as med_assemble_h by quarter
| eval deficiency_rate=round(100*deficient/packs,1)
| sort quarter
```
