# 22.31.15 — Control testing compliance evidence

**Criticality:** low | **Difficulty:** expert | **Status:** N/A

## Description

Evidences APRA CPS 234 obligations for control testing compliance evidence using endpoint, identity, and cloud telemetry.

## Value

Gives the responsible team auditor-ready evidence for APRA CPS 234 compliance by automating regulatory compliance evidence using indexes test and sourcetypes cps234:control_testing. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=test sourcetype="cps234:control_testing" earliest=-365d
| stats latest(effective) as e by control_id
| where e="false"
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Endpoint.Processes by Processes.process_name, Processes.user, Processes.dest | sort - count
```
