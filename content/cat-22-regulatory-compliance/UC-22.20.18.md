# 22.20.18 — CMMC assessment readiness — artifact 3

**Criticality:** medium | **Difficulty:** expert | **Status:** N/A

## Description

Demonstrates contractor cybersecurity hygiene for "CMMC assessment readiness — artifact 3" using CUI-scoped telemetry mapped to CMMC 2.0 Assessment expectations.

## Value

Gives the responsible team auditor-ready evidence for CMMC compliance by automating regulatory compliance evidence using indexes grc and sourcetypes cmmc:gap. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=grc sourcetype=cmmc:gap earliest=-90d
| eval uc_tag="22.20.18"
| where gap_severity="high" AND remediation_owner="unassigned"
| table gap_id, practice_id, due_date
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Endpoint.Processes by Processes.process_name, Processes.user, Processes.dest | sort - count
```

## Implementation

(1) Confirm field extractions and CIM tags for the sourcetypes in scope; (2) Load or maintain the referenced lookups/KVStore collections with owner attestation dates; (3) Schedule the search at an interval aligned to the control’s materiality; (4) Route positive findings to the compliance ticketing queue with required evidence fields; (5) Retain scheduled search exports per records management and legal hold procedures. Indexes required: `index=grc`.

## Visualization

Stacked bar, Table, Time chart, Single value. Key fields: `due_date`, `gap_id`, `practice_id`, `uc_tag`.
