# 22.15.22 — OT Auditable Event Coverage by Zone and Source

**Criticality:** high | **Difficulty:** intermediate | **Status:** verified

## Description

Compares expected auditable sources per IEC 62443 zone to actual indexed event rates to find blind spots.

## Value

Gives the responsible team auditor-ready evidence for IEC 62443 compliance by automating regulatory compliance evidence using indexes ot_audit. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
| tstats count where index=ot_audit by sourcetype, host prefix=ot
| lookup ot_expected_audit_sources.csv sourcetype OUTPUT zone, required
| eval gap=if(required="true" AND count<100,"low_volume","ok")
| where gap!="ok"
| table sourcetype, host, count, zone
```

## Implementation

(1) Define required sourcetypes per Purdue level; (2) onboard missing forwarders; (3) validate license throttling not dropping; (4) quarterly coverage attestation. Indexes required: `index=ot_audit`; Lookups: `ot_expected_audit_sources`, `ot_expected_audit_sources.csv`.

## Visualization

Bar (actual vs expected), Table, Single value (gaps).
