# UC-22.11.73 · ASV External Scan Failure and Non-Compliant Finding Trend

## Description

Tracks approved scanning vendor run outcomes and residual high-risk findings on Internet-facing payment hosts so quarterly external vulnerability scanning evidence stays current and remediated.

## Value

Gives the responsible team auditor-ready evidence for PCI DSS compliance by automating regulatory compliance evidence using indexes vm and sourcetypes qualys:was. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Tag ASV scans distinctly from internal VM. (2) Join open findings `severity>=3` on same `fqdn`. (3) Alert if `fail_pct>0` two quarters in a row. (4) Store PDF reports in GRC with Splunk deep link. Indexes required: `index=vm`; Sourcetypes: `sourcetype=qualys:was`.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Add-on for Qualys (2964), Splunk Add-on for Tenable (4060).
• Ensure the following data sources are available: `index=vm` `qualys:was` or `tenable:io:vm` with `scan_type="ASV"` if tagged.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Tag ASV scans distinctly from internal VM. (2) Join open findings `severity>=3` on same `fqdn`. (3) Alert if `fail_pct>0` two quarters in a row. (4) Store PDF reports in GRC with Splunk deep link.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=vm sourcetype="qualys:was" earliest=-120d
| where match(scan_name,"(?i)ASV|PCI\\s*External)")
| eval failed_scan=if(match(status,"(?i)error|failed|aborted"),1,0)
| timechart span=30d sum(failed_scan) as failed_runs, count as total_runs
| eval fail_pct=if(total_runs>0, round(100*failed_runs/total_runs,2), null())
```
