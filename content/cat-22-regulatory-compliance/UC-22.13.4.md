# 22.13.4 — Annual CIP-002 Categorization Review Evidence Package

**Criticality:** high | **Difficulty:** intermediate | **Status:** N/A

## Description

We timestamp inventory validation searches and signed review workflows so annual categorization review evidence is searchable and complete.

## Value

Gives the responsible team auditor-ready evidence for NERC CIP compliance by automating regulatory compliance evidence using indexes cip_evidence, itsm and sourcetypes cip:review_ticket, cip:review_ticket`, cip:search_evidence. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=itsm sourcetype=cip:review_ticket review_type="CIP-002_Categorization" earliest=-400d
| eval signed_epoch=strptime(signed_at,"%Y-%m-%dT%H:%M:%SZ")
| eval annual_due=relative_time(now(),"-1y@d")
| eval signed_ok=if(signed_epoch>=annual_due,1,0)
| append [
    search index=cip_evidence sourcetype=cip:search_evidence search_name="CIP002_inventory_recon" earliest=-400d
    | stats latest(_time) as last_proof by search_name
  ]
| table review_type, owner, signed_at, signed_ok, last_proof
```

## Implementation

(1) Emit HEC when the responsible manager signs the annual review. (2) Schedule the inventory recon search to write `cip:search_evidence` events. (3) Alert if `signed_ok=0` approaching the compliance date. (4) PDF the dashboard for the audit binder. Indexes required: `index=cip_evidence`, `index=itsm`; Sourcetypes: `sourcetype=cip:review_ticket`, `sourcetype=cip:review_ticket``, `sourcetype=cip:search_evidence`, `sourcetype=cip:search_evidence``.

## Visualization

Single value (review current), Timeline (signatures vs proof runs), Table (owners)
