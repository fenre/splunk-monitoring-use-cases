# UC-22.15.19 · Unsigned Macros or Scripts Executed on Engineering Workstation

## Description

Monitors execution of script engines and office macros from OT engineering hosts where mobile code should be restricted.

## Value

Gives the responsible team auditor-ready evidence for IEC 62443 compliance by automating regulatory compliance evidence using indexes sysmon. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Deploy Sysmon with tuned config for EWS; (2) Approve known vendor installers via hash lookup; (3) block VBA macros at policy layer; (4) review weekly top commands. Indexes required: `index=sysmon`.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Add-on for Microsoft Windows (742).
• Ensure the following data sources are available: `WinEventLog:Microsoft-Windows-Sysmon/Operational` EventCode 1.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Deploy Sysmon with tuned config for EWS; (2) Approve known vendor installers via hash lookup; (3) block VBA macros at policy layer; (4) review weekly top commands.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=sysmon EventCode=1 host IN ("OT-EWS-*") earliest=-7d
| where (Image LIKE "%\powershell.exe" OR Image LIKE "%\mshta.exe" OR Image LIKE "%\cscript.exe")
    AND NOT(CommandLine LIKE "%-nop -enc%")
| lookup ews_signed_scripts.csv CommandLine OUTPUT approved
| where approved!="true"
| stats count by host, Image, CommandLine
| sort - count
```
