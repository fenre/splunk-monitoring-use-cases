# UC-22.14.51 · Integrated Information Security Analysis Team Handoffs

## Description

Tracks cross-team comments and assignments between SOC and CTI for IR-10 integrated infosec analysis team collaboration.

## Value

Tracks cross-team comments and assignments between SOC and CTI for IR-10 integrated infosec analysis team collaboration.

## Implementation

(1) Validate `transaction` performance — limit indexes; (2) alternatively use `streamstats` on sorted events; (3) KPI for handoff latency; (4) monthly leadership review.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Add-on for ServiceNow (Splunkbase 1928).
• Ensure the following data sources are available: `snow:incident` with assignment_group transitions.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Validate `transaction` performance — limit indexes; (2) alternatively use `streamstats` on sorted events; (3) KPI for handoff latency; (4) monthly leadership review.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=itsm sourcetype=snow:incident category="Security" earliest=-30d
| transaction number maxspan=24h startswith="assignment_group=SOC" endswith="assignment_group=CTI"
| where duration>3600
| table number, duration, assignment_group, short_description
```
