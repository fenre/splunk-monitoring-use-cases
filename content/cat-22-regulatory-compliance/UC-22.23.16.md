Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=product_security (sourcetype="sec:incident:notify" OR sourcetype="github:actions") earliest=-30d
| eval cra_tag="enisa"
| eval uc_id="22.23.16"
| stats count as events, sum(eval(status="fail" OR result="fail")) as fails, dc(release_id) as releases by product_id, cra_tag
| where fails>0
| sort - fails
| head 200
```
