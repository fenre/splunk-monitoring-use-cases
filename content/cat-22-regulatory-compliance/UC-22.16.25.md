# 22.16.25 — SCADA master redundancy failover duration

**Criticality:** high | **Difficulty:** beginner | **Status:** N/A

## Description

Supports pipeline cybersecurity program oversight under TSA Pipeline Security Directive by correlating operational telemetry with governance records for "SCADA master redundancy failover duration".

## Value

Gives the responsible team auditor-ready evidence for TSA SD compliance by automating regulatory compliance evidence using indexes scada and sourcetypes ignition:audit. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=scada sourcetype=ignition:audit earliest=-30d
| eval uc_tag="22.16.25"
| where match(message,"(?i)failover|redundancy")
| stats count by redundant_pair, host
```

## CIM SPL
