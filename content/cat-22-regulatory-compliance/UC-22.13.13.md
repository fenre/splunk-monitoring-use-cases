# 22.13.13 — Quarterly Access Review Dataset — Interactive Sessions by System

**Criticality:** high | **Difficulty:** intermediate | **Status:** N/A

## Description

We assemble quarterly interactive access by user and BES asset to accelerate access certification and attestation.

## Value

Gives the responsible team auditor-ready evidence for NERC CIP compliance by automating regulatory compliance evidence using indexes pam and sourcetypes cyberark:ptalog, cyberark:ptalog`. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=pam sourcetype=cyberark:ptalog action="Logon" earliest=-90d@d latest=@d
| lookup bes_cyber_asset_inventory.csv address AS address OUTPUT bes_system impact_rating
| where isnotnull(bes_system)
| stats earliest(_time) as first latest(_time) as last dc(address) as targets by user, bes_system, impact_rating
| sort impact_rating, user
```
