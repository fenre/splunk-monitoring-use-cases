# 22.16.28 — Monthly OT security posture score trend by site

**Criticality:** critical | **Difficulty:** intermediate | **Status:** N/A

## Description

Supports pipeline cybersecurity program oversight under TSA Pipeline Security Directive by correlating operational telemetry with governance records for "Monthly OT security posture score trend by site".

## Value

Gives the responsible team auditor-ready evidence for TSA SD compliance by automating regulatory compliance evidence using indexes summary and sourcetypes ot:posture:monthly. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=summary sourcetype=ot:posture:monthly earliest=-400d
| eval uc_tag="22.16.28"
| timechart span=1mon avg(score) by site_id
```

## CIM SPL

```spl
| tstats summariesonly=t avg(All_Traffic.bytes_in) as agg_value from datamodel=Network_Traffic.All_Traffic by All_Traffic.action, All_Traffic.src, All_Traffic.dest, All_Traffic.dest_port span=1mon | sort - agg_value
```
