# 22.37.1 — Consent capture evidence freshness — stale-consent alerting

**Criticality:** high | **Difficulty:** intermediate | **Status:** verified

## Description

Flags subjects whose consent is older than the program threshold yet whose data is still being processed. Stale consent is a lawful-basis failure — Art.6/Art.7 require consent to remain current and demonstrable.

## Value

Prevents the 'set-and-forget' consent anti-pattern that surfaces in nearly every GDPR supervisory-authority enforcement action.

## SPL

```spl
index=cmp sourcetype=onetrust:consent earliest=-400d
| stats max(_time) as last_consent by subject_token purpose
| eval consent_age_days=round((now()-last_consent)/86400,0)
| where consent_age_days>365
| join subject_token [search index=app_activity earliest=-30d | stats count by subject_token | where count>0]
| table subject_token purpose last_consent consent_age_days
```

## Implementation

(1) Ingest CMP events with purpose taxonomy; (2) set max_consent_age_days via macro; (3) run daily; (4) pipe to SOAR to drive a re-consent workflow. Indexes required: `index=app_activity`, `index=cmp`; Sourcetypes: `sourcetype=onetrust:consent`.

## Visualization

Histogram of consent age by purpose; table of stale subjects; single-value for '% of processing with fresh consent'.

## Compliance

- **GDPR** 2016/679: Art.7 (partial) — Art.7 conditions for consent — the UC evidences when consent was captured and surfaces subjects whose consent is older than the policy threshold.
- **GDPR** 2016/679: Art.6 (partial) — Art.6(1)(a) lawful basis of consent requires demonstrable consent. Missing or stale evidence means no lawful basis.
- **UK GDPR** post-Brexit: Art.7 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.7.
- **UK GDPR** post-Brexit: Art.6 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.6.

## Known False Positives

Long-lived contractual relationships where consent is not the lawful basis. Use a lawful_basis.csv join to filter them out.
