# UC-22.7.46 · Scheduled Restore Test Outcomes vs Policy Frequency

## Description

Step 1 — Configure data collection
(1) Align to RTO tiers; (2) document failures; (3) executive attestation.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=backup sourcetype=cohesity:restore_test earliest=-365d
| stats latest(status) as last_status, latest(_time) as last_run by policy_name
| eval days_since=(now()-last_run)/86400
| where days_since>90 OR last_status!="success"
```
