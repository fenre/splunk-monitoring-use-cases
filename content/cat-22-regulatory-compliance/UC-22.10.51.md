# UC-22.10.51 · Business Associate Access — VPN/SSO Sessions Originating from BA Address Space

## Description

Surfaces new source ASNs or IP ranges used by business associates connecting into hosted EHR jump boxes so BA access stays within contracted technical boundaries.

## Value

Surfaces new source ASNs or IP ranges used by business associates connecting into hosted EHR jump boxes so BA access stays within contracted technical boundaries.

## Implementation

(1) Define `ba_ip_ranges` in `transforms.conf` as a CIDR lookup on `client_ip`, or maintain one row per approved egress IP in `ba_ip_ranges.csv`; (2) refresh when BA contracts change VPN concentrators; (3) alert on first-seen IP before allowlisting; (4) optional threat intel enrichment on `client_ip`.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Add-on for Citrix (2757), Splunk Enterprise Security (263).
• Ensure the following data sources are available: `index=citrix` `sourcetype=citrix:session`, CIDR lookup `ba_ip_ranges` in `transforms.conf` (`vendor_name`, `baa_id`) or `ba_ip_ranges.csv` with one row per approved egress IP.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Define `ba_ip_ranges` in `transforms.conf` as a CIDR lookup on `client_ip`, or maintain one row per approved egress IP in `ba_ip_ranges.csv`; (2) refresh when BA contracts change VPN concentrators; (3) alert on first-seen IP before allowlisting; (4) optional threat intel enrichment on `client_ip`.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=citrix sourcetype="citrix:session" earliest=-7d PublishedApplication="*Epic*BAA*"
| lookup ba_ip_ranges client_ip OUTPUT vendor_name, baa_id
| where isnull(vendor_name)
| iplocation client_ip
| stats count dc(UserName) as distinct_users by client_ip, Country, Org, PublishedApplication
| sort - count
```
