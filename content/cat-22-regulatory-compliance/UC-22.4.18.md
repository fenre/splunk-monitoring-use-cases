# UC-22.4.18 · CCPA/CPRA — Minor Consent — Marketing Cookie Fires Before Parental Consent Timestamp

## Description

Tracking technologies on minor-directed experiences require heightened controls. Tag-manager events that load ad or analytics pixels prior to recorded parental consent timestamps evidence misconfiguration of consent mode integrations.

## Value

Tracking technologies on minor-directed experiences require heightened controls. Tag-manager events that load ad or analytics pixels prior to recorded parental consent timestamps evidence misconfiguration of consent mode integrations.

## Implementation

(1) Scope to minor-designated site IDs via lookup `minor_site_ids.csv`; (2) partner with marketing ops on tag inventory; (3) GPC and CMP linkage testing in pre-prod; (4) document vendor contracts; (5) evidence for AG inquiries on dark pattern adjacent issues.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=web sourcetype="consent:web" earliest=-30d
| eval pc=strptime(parental_consent_ts,"%Y-%m-%dT%H:%M:%SZ")
| join type=left session_id [
    search index=web sourcetype="tag:fire" marketing_flag=true earliest=-30d
    | fields session_id, tag_vendor, _time
  ]
| where _time < pc OR isnull(pc)
| stats count by tag_vendor, session_id
| sort - count
```
