# UC-22.3.27 · DORA TLPT — Findings Severity, Remediation Owner, and Due Date Tracking

## Description

TLPT produces material findings. Monitoring remediation SLAs by severity with linkage to ICT third-party providers evidences closed-loop treatment required for resilience testing programs.

## Value

TLPT produces material findings. Monitoring remediation SLAs by severity with linkage to ICT third-party providers evidences closed-loop treatment required for resilience testing programs.

## Implementation

(1) Map severity to internal SLAs; (2) auto-create problems in ITSM; (3) escalate critical past-due to management body; (4) join provider_id to RoI; (5) quarterly export for regulator file.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=grc sourcetype="tlpt:finding" status!="closed" earliest=-400d
| eval due=strptime(due_date,"%Y-%m-%d")
| where due<now() OR isnull(owner)
| stats count by severity, related_provider_id, owner
| sort severity, - count
```
