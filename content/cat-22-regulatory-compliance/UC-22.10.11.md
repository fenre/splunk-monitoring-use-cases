# UC-22.10.11 ┬Ę Business Associate Agreements ŌĆö Expiry & Auto-Renewal Monitoring

## Description

Prevents lapsed BAAs for vendors that process ePHI by tracking expiry, auto-renew flags, and required amendment workflows.

## Value

Prevents lapsed BAAs for vendors that process ePHI by tracking expiry, auto-renew flags, and required amendment workflows.

## Implementation

(1) Nightly refresh from CLM API; (2) route CRITICAL to legal ops; (3) map vendor_id to technical assets for coverage attestation.

Step 2 ŌĆö Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
| inputlookup baa_master.csv
| eval days_to_end=round((strptime(baa_end,"%Y-%m-%d")-now())/86400,0)
| eval alert_tier=case(days_to_end<0,"EXPIRED",days_to_end<=30,"CRITICAL",days_to_end<=90,"WARN",1=1,"OK")
| where alert_tier!="OK"
| table vendor_id, vendor_name, baa_end, auto_renew, alert_tier, data_categories
| sort days_to_end
```
