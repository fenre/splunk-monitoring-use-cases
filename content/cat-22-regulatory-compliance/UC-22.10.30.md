# UC-22.10.30 ┬Ę Media Controls ŌĆö Large PHI Print Jobs to Non-Secure Printers

## Description

Monitors unusually large chart print jobs routed to non-secure print queues so physical disclosure risks from uncontrolled hardcopy are reduced.

## Value

Monitors unusually large chart print jobs routed to non-secure print queues so physical disclosure risks from uncontrolled hardcopy are reduced.

## Implementation

(1) Classify printers by site/zone; (2) integrate Pharos/Equitrac if used; (3) educate on minimum necessary printing.

Step 1 ŌĆö Configure data collection

Step 2 ŌĆö Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=windows sourcetype="WinEventLog:Microsoft-Windows-PrintService/Operational" EventCode="307" earliest=-7d
| eval pages=tonumber(Pages)
| lookup secure_print_queues.csv PrinterName OUTPUT secure_queue
| where (isnull(secure_queue) OR secure_queue="false") AND pages>=50
| stats sum(pages) as total_pages count as jobs by UserName, PrinterName
| sort - total_pages
```
