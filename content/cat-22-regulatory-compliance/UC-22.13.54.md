# 22.13.54 — Removable Media Mount Events on Engineering Workstations

**Criticality:** high | **Difficulty:** intermediate | **Status:** N/A

## Description

We detect USB or removable media usage on BES engineering workstations to enforce transient media controls.

## Value

Gives the responsible team auditor-ready evidence for NERC CIP compliance by automating regulatory compliance evidence using indexes wineventlog and sourcetypes WinEventLog:Security. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=wineventlog sourcetype="WinEventLog:Security" EventCode=6416 earliest=-7d
| lookup bes_engineering_hosts.csv ComputerName OUTPUT bes_system
| where isnotnull(bes_system)
| stats count by ComputerName, DeviceName, ClassName, bes_system
| sort - count
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Endpoint.Processes by Processes.process_name, Processes.user, Processes.dest | sort - count
```

## Implementation

(1) Enable device installation auditing. (2) Optionally deploy Sysmon for richer context. (3) Weekly review with OT engineering. (4) Document approved maintenance media in lookup. Indexes required: `index=wineventlog`; Sourcetypes: `sourcetype=WinEventLog:Security`.

## Visualization

Table (mounts), Bar chart (by host), Single value (distinct devices). Key fields: `ClassName`, `ComputerName`, `DeviceName`, `bes_system`.

## Compliance

- **NERC CIP** current: CIP-010-4 R6 Part 6.1 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for CIP-010-4 R6 Part 6.1.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
