Step 2 Рђћ Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
`notable` HIPAA_Breach_Candidate=1 earliest=-90d
| eval detect_time=_time
| join IncidentId type=left [
    search index=dlp sourcetype="o365:dlp" earliest=-90d
    | eval first_indicator=strptime(CreatedTime,"%Y-%m-%dT%H:%M:%SZ")
    | stats min(first_indicator) as first_dlp by IncidentId ]
| eval hours_to_detect=round((detect_time-first_dlp)/3600,2)
| table IncidentId, first_dlp, detect_time, hours_to_detect, rule_name, status
| sort - hours_to_detect
```
