# UC-22.11.23 · Weak Cipher Suites Offered by Internal TLS Terminators

## Description

Highlights servers advertising RC4, 3DES, or NULL ciphers on CDE-facing services, supporting the strong cryptography requirement for transmissions of PAN and connection data.

## Value

Highlights servers advertising RC4, 3DES, or NULL ciphers on CDE-facing services, supporting the strong cryptography requirement for transmissions of PAN and connection data.

## Implementation

(1) Ingest Stream metadata from SPAN or tap on DMZ switches. (2) Refresh listener inventory when load balancers change. (3) Pair with Qualys SSL scan for corroboration. (4) Track remediation SLA in ITSM.

Prerequisites
• Install and configure the required add-on or app: Splunk Stream (1809), Splunk Add-on for Qualys (2964).

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=stream sourcetype="stream:tls" earliest=-24h
| lookup pci_tls_terminators.csv server_ip OUTPUT pci_listener
| where pci_listener="true"
| where match(cipher_suite,"(?i)RC4|3DES|NULL|EXPORT|anon")
| stats count by server_ip, cipher_suite, tls_version
| sort - count
```
