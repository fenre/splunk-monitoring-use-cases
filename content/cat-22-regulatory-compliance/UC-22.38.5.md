# 22.38.5 — Bulk regulated-data export targeting non-adequate jurisdiction

**Criticality:** critical | **Difficulty:** advanced | **Status:** draft

## Description

Intersects bulk export operations (DB dumps, large cloud uploads) with destination geolocation and the sanctioned-transfer catalogue. Bulk exports targeting non-adequate jurisdictions without SCCs are a category of breach that requires immediate executive escalation.

## Value

Turns post-hoc 'where did our customer dataset go?' incident forensics into real-time prevention.

## SPL

```spl
index=db_audit action=export OR (index=cloud_storage action=PutObject bytes>1073741824) earliest=-10m
| lookup data_classification.csv object_key OUTPUT classification regulated_scope
| where isnotnull(regulated_scope)
| `get_geolocation(dest_ip)`
| lookup sanctioned_transfer_jurisdictions.csv country as dest_country OUTPUT scc_active adequacy_decision
| where isnull(scc_active) AND isnull(adequacy_decision)
| table _time user object_key classification regulated_scope dest_country bytes
```

## Implementation

(1) Ensure DB and cloud storage layers emit size + destination; (2) maintain classification and jurisdiction lookups; (3) schedule every 10 min; (4) SOAR auto-blocks destination if exception policy permits. Indexes required: `index=cloud_storage`, `index=db_audit`.

## Visualization

World-map of bulk exports, table of findings with destination geography, single-value 'bulk exports within sanctioned geographies'. Key fields: `bytes`, `classification`, `dest_country`, `object_key`, `regulated_scope`, `user`.

## Compliance

- **GDPR** 2016/679: Art.44 (full) — Art.44 general principle for transfers is directly enforced by blocking bulk exports without safeguards.
- **GDPR** 2016/679: Art.45 (partial) — Art.45 adequacy decision list membership is directly queried.
- **DORA** Regulation (EU) 2022/2554: Art.28 (partial) — DORA Art.28 ICT third-party oversight implicates cross-border data dependencies.
- **UK GDPR** post-Brexit: Art.44 (partial) — Derived from GDPR@2016/679 Art.44 (parent assurance='full'). UK GDPR preserves parent clause numbering; inherited assurance degraded one step.
- **UK GDPR** post-Brexit: Art.45 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.45.

## Known False Positives

Bona-fide DR or migration operations authorised under a dedicated exception ID must be logged to a migrations.csv allow-list.
