# 22.31.7 — Microsoft Office macro security baseline compliance

**Criticality:** critical | **Difficulty:** expert | **Status:** N/A

## Description

Evidences ASD Essential Eight obligations for microsoft office macro security baseline compliance using endpoint, identity, and cloud telemetry.

## Value

Gives the responsible team auditor-ready evidence for ASD E8 compliance by automating regulatory compliance evidence using indexes winregistry and sourcetypes WinRegistry. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=winregistry sourcetype="WinRegistry" reg_path="*Office*Security*VBAWarnings*" earliest=-7d
| stats latest(reg_value_data) as v by host
| where v!="3"
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Endpoint.Processes by Processes.dest | sort - count
```

Indexes required: `index=winregistry`; Sourcetypes: `sourcetype=WinRegistry`.
