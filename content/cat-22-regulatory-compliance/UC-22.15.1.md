# UC-22.15.1 · OT Security Policy Control Evidence from Log Review

## Description

Correlates policy-required controls (access, change, remote sessions) to indexed evidence so the IACS security program can demonstrate ongoing compliance with documented requirements.

## Value

Gives the responsible team auditor-ready evidence for IEC 62443 compliance by automating regulatory compliance evidence using indexes ot_sec and sourcetypes claroty:alert, nozomi:alerts, pan:system. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Maintain lookup mapping control families to 2-1 policy clauses; (2) normalize sourcetypes with CIM tags where applicable; (3) schedule weekly PDF for program reviews; (4) route uncategorized families to GRC. Indexes required: `index=ot_sec`.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk OT Security Add-on (5151), Splunk Enterprise Security (263).
• Ensure the following data sources are available: `index=ot_sec` — `sourcetype="nozomi:alerts"`, `sourcetype="claroty:alert"`, `sourcetype="pan:system"`; lookup `iec62443_policy_controls.csv`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Maintain lookup mapping control families to 2-1 policy clauses; (2) normalize sourcetypes with CIM tags where applicable; (3) schedule weekly PDF for program reviews; (4) route uncategorized families to GRC.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ot_sec earliest=-30d (sourcetype="nozomi:alerts" OR sourcetype="claroty:alert" OR sourcetype="pan:system")
| eval control_family=case(match(_raw,"(?i)remote"),"Remote_Access", match(_raw,"(?i)change|config"),"Change_Management", true(),"General_Security_Event")
| lookup iec62443_policy_controls.csv control_family OUTPUT policy_clause
| stats count by control_family, policy_clause, sourcetype
| sort - count
```
