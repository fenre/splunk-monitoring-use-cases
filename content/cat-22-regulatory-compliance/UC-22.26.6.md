# UC-22.26.6 · Power system availability monitoring

## Description

Correlates SCADA-linked outages and restorations to evidence availability monitoring for electricity preparedness.

## Value

Correlates SCADA-linked outages and restorations to evidence availability monitoring for electricity preparedness.

## Implementation

(1) Maintain supporting lookups (classified hosts, sectors, transfers) aligned to Norwegian regulatory language. (2) Schedule searches with Europe/Oslo time semantics for reporting cutoffs. (3) Route ES notables to the correct national security, OT, or privacy queues. (4) Quarterly false-positive tuning with HR, operations, and legal stakeholders.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ot sourcetype IN ("opcua:session","scada:hmi:login") asset="GRID-*" earliest=-24h
| bin span=15m _time
| stats dc(session_id) as sessions by _time, asset, user
| eventstats median(sessions) as med by asset
| where sessions > med*3 AND sessions>4
| sort - sessions
```
