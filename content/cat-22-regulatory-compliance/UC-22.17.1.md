# 22.17.1 — Electronic records integrity — control theme 1

**Criticality:** high | **Difficulty:** beginner | **Status:** N/A

## Description

Supports Part 11 technical controls evidence by correlating validated system audit trails with identity and change records.

## Value

Supports Part 11 technical controls evidence for "Electronic records integrity — control theme 1" by correlating validated system audit trails with identity and change records.

## SPL

```spl
index=gxp sourcetype=lims:audit earliest=-24h action IN ("INSERT","UPDATE","DELETE")
| eval uc_tag="22.17.1"
| eval gap=if(isnull(reason_code) AND action!="INSERT",1,0)
| stats sum(gap) as missing_reason, count by instrument_id, user_id
| where missing_reason>0
```
