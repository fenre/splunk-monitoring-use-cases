# UC-22.14.62 · Plan of Action and Milestones Open Items Past Due

## Description

Lists overdue POA&M items for CA-5 plans of action and milestones.

## Value

Gives the responsible team auditor-ready evidence for NIST 800-53 compliance by automating regulatory compliance evidence. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Automate CSV refresh from authoritative GRC; (2) RBAC the lookup; (3) weekly leadership digest; (4) integrate CAP tracking.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk DB Connect (Splunkbase 2686) or CSV lookup updated by GRC.
• Ensure the following data sources are available: `inputlookup poam_open.csv`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Automate CSV refresh from authoritative GRC; (2) RBAC the lookup; (3) weekly leadership digest; (4) integrate CAP tracking.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
| inputlookup poam_open.csv
| eval due_epoch=strptime(due_date,"%Y-%m-%d")
| where due_epoch < now() AND status!="Closed"
| table poam_id, control, owner, due_date, status, risk_rating
| sort due_date
```
