# UC-22.13.6 · Cyber Security Plan Technical Control Attestation — MFA for Privileged Access

## Description

We compare multifactor authentication signals for privileged BES accounts against the cyber security plan so documented controls match authentication reality.

## Value

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=wineventlog sourcetype="WinEventLog:Security" EventCode=4624 earliest=-24h
| lookup bes_privileged_users.csv Account_Name AS user OUTPUT requires_mfa bes_system
| where requires_mfa="true"
| eval strong_pkg=if(match(AuthenticationPackageName,"(?i)Negotiate|Kerberos") AND LogonType IN ("10","3"),1,0)
| stats count as logons sum(strong_pkg) as strong_signals by user, bes_system
| eval ratio=round(strong_signals/logons,3)
| where ratio < 0.9
```
