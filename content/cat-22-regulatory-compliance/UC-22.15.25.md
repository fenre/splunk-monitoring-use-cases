# UC-22.15.25 ¬∑ Clock Skew on OT Hosts vs NTP Stratum

## Description

Surfaces OT servers and RTUs with large `_indextime` vs `_time` deltas undermining trustworthy audit timestamps.

## Value

Surfaces OT servers and RTUs with large `_indextime` vs `_time` deltas undermining trustworthy audit timestamps.

## Implementation

(1) Enforce PTP/NTP on all IEDs; (2) fix dual-homed syslog paths; (3) alert >120s; (4) document stratum hierarchy per site.

Step 2 ‚Äî Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ot_infra earliest=-24h
| eval skew=abs(_indextime-_time)
| where skew>120
| stats max(skew) as max_skew, perc95(skew) as p95_skew by host, sourcetype
| sort - p95_skew
```
