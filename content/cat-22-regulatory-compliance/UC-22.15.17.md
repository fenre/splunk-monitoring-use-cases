# 22.15.17 — OT Access from Untrusted Networks Without Split Tunnel Block

**Criticality:** critical | **Difficulty:** intermediate | **Status:** N/A

## Description

Flags VPN or agent sessions originating from high-risk geos or residential ASNs while accessing OT jump portals.

## Value

Gives the responsible team auditor-ready evidence for IEC 62443 compliance by automating regulatory compliance evidence using indexes vpn and sourcetypes pan:globalprotect. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=vpn sourcetype="pan:globalprotect" earliest=-7d
| iplocation public_ip
| lookup ot_trusted_country.csv Country OUTPUT trust
| where trust!="trusted" OR like(ASN,"*RESIDENTIAL*")
| lookup ot_jump_portals.csv portal OUTPUT sl_t
| stats earliest(_time) as first, latest(_time) as last, values(public_ip) as ips by user, portal, Country
| sort - last
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Network_Traffic.All_Traffic by All_Traffic.action, All_Traffic.src, All_Traffic.dest, All_Traffic.dest_port | sort - count
```

## Implementation

(1) Maintain country allowlist aligned to business; (2) require hardware token from untrusted regions; (3) integrate threat intel ASN feed; (4) auto-disable accounts on repeat hits. Indexes required: `index=vpn`; Sourcetypes: `sourcetype=pan:globalprotect`; Lookups: `ot_jump_portals`, `ot_jump_portals.csv`, `ot_trusted_country`, `ot_trusted_country.csv`.

## Visualization

Choropleth, Table, Single value (sessions). Key fields: `Country`, `first`, `ips`, `last`, `portal`, `user`.

## Compliance

- **IEC 62443** 2013-ongoing: SR 1.13 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for SR 1.13.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
