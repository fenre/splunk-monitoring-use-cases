# UC-22.11.69 · Automated Log Review — Correlation of Firewall Deny Bursts with IDS Signatures

## Description

Joins short-window firewall denies with IDS alerts touching the same `src_ip` so automated reviews surface coordinated attack patterns without waiting for manual cross-index pivoting.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=netfw sourcetype="pan:traffic" action=deny earliest=-1h
| stats count by src, dest
| where count>500
| join type=left max=0 src [
    search index=ids (sourcetype="suricata:alert" OR sourcetype="pan:threat") earliest=-1h
    | rename src_ip as src
    | stats dc(signature) as sigs by src
  ]
| where isnotnull(sigs) AND sigs>0
| table src, dest, count, sigs
```
