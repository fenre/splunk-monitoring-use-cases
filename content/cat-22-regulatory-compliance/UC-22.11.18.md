# UC-22.11.18 · PAN Masking Validation in Point-of-Sale and Web Receipt Logs

## Description

Confirms that customer-visible channels emit only first six and last four digits, catching accidental logging of full PAN in receipt or audit trails.

## Value

Confirms that customer-visible channels emit only first six and last four digits, catching accidental logging of full PAN in receipt or audit trails.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
(index=pos sourcetype="pos:receipt") OR (index=web sourcetype="access_combined") earliest=-24h
| where match(_raw,"\\b[0-9]{13,19}\\b")
| where NOT match(_raw,"\\b[0-9]{6}[\\*Xx-]{3,}[0-9]{4}\\b")
| stats count by index, sourcetype, host, uri
| sort - count
```
