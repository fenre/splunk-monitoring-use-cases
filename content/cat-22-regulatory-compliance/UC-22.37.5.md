# 22.37.5 — IAB TCF consent string mutation without user interaction

**Criticality:** high | **Difficulty:** advanced | **Status:** draft

## Description

Compares the TCF 2.2 consent string across consecutive page impressions in the same session and flags changes that occurred without a corresponding CMP interaction within 60 seconds. This is a known pattern of rogue ad-tech overriding user choice.

## Value

Catches a pattern that has surfaced repeatedly in EDPB enforcement decisions (Meta, Google ad-tech) before it becomes a supervisory finding.

## SPL

```spl
index=web sourcetype=access_combined earliest=-1h euconsent_v2=*
| sort 0 session_id _time
| streamstats current=f last(euconsent_v2) as prior_consent by session_id
| where isnotnull(prior_consent) AND euconsent_v2!=prior_consent
| join type=outer session_id _time [search index=cmp sourcetype=onetrust:cmp action=interaction earliest=-1h | fields session_id _time | rename _time as interaction_time]
| where isnull(interaction_time) OR abs(_time-interaction_time)>60
| table _time session_id prior_consent euconsent_v2
```

## Implementation

(1) Preserve Set-Cookie headers in access logs; (2) ingest CMP interaction events; (3) run hourly; (4) on hit, freeze the vendor via ad-tech allow-list and open a DPO ticket. Indexes required: `index=cmp`, `index=web`; Sourcetypes: `sourcetype=access_combined`, `sourcetype=onetrust:cmp`.

## Visualization

Table of mutating sessions by vendor, bar chart of mutations per ad-tech partner, single-value 'sessions with genuine consent (24h)'. Key fields: `euconsent_v2`, `interaction_time`, `prior_consent`, `session_id`.

## Compliance

- **GDPR** 2016/679: Art.7(1) (partial) — Art.7(1) requires the controller to demonstrate consent; detecting silent string mutation is one part of that demonstration.
- **CCPA** CPRA (as amended): §1798.135(b) (contributing) — CPRA opt-out signals (GPC) can be misrepresented by server-side mutation; detection adds evidentiary weight.
- **UK GDPR** post-Brexit: Art.7(1) (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.7(1).

## Known False Positives

CMP token refreshes (same purpose mask, new expiry) can legitimately rewrite the TC string; allow-list based on purpose-bitmap equality.
