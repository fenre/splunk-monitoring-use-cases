# UC-22.7.36 · Lateral Movement Chain — Auth, RDP, and Process Create Same Src

## Description

Joins authentication, RDP, and process events within a window for adverse event analysis.

## Value

Joins authentication, RDP, and process events within a window for adverse event analysis.

## Implementation

(1) Tune for VDI noise; (2) map to ATT&CK; (3) auto-notable optional.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Windows Security, Sysmon.
• Ensure the following data sources are available: `WinEventLog:Security`, Sysmon.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Tune for VDI noise; (2) map to ATT&CK; (3) auto-notable optional.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=windows earliest=-24h (sourcetype=WinEventLog:Security EventCode=4624 OR sourcetype=XmlWinEventLog:Microsoft-Windows-Sysmon/Operational EventCode=1)
| eval src=coalesce(src_ip, IpAddress)
| transaction src maxspan=300s maxevents=50
| where mvcount(EventCode)>2
```
