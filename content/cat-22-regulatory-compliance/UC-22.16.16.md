# 22.16.16 — Regulatory notification task checklist aging

**Criticality:** critical | **Difficulty:** intermediate | **Status:** N/A

## Description

Supports pipeline cybersecurity program oversight under TSA Pipeline Security Directive by correlating operational telemetry with governance records for "Regulatory notification task checklist aging".

## Value

Gives the responsible team auditor-ready evidence for TSA SD compliance by automating regulatory compliance evidence using indexes itsm and sourcetypes snow:sc_req_item. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=itsm sourcetype=snow:sc_req_item short_description="*TSA*notification*" earliest=-365d
| eval uc_tag="22.16.16"
| eval opened=strptime(opened_at,"%Y-%m-%d %H:%M:%S")
| where isnull(closed_at)
| stats count by cat_item, assignment_group
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Network_Traffic.All_Traffic by All_Traffic.action, All_Traffic.src, All_Traffic.dest, All_Traffic.dest_port | sort - count
```
