# UC-22.7.44 · Legal Hold Population — Elevated File Export Activity

## Description

Tracks downloads for users on legal hold to align insider communication risk with coordinated response procedures.

Step 1 — Configure data collection
(1) Coordinate with legal; (2) strict access to dashboard; (3) align to RS.CO procedures.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=o365 sourcetype=ms:o365:management Operation=FileDownloaded earliest=-30d
| lookup legal_hold_users.csv UserPrincipalName OUTPUT on_hold
| where on_hold="true"
| bucket _time span=1d
| stats count by UserPrincipalName, _time
```
