# 22.34.3 — Physical and logical access correlation for SWIFT operators

**Criticality:** critical | **Difficulty:** expert | **Status:** N/A

## Description

Produces SWIFT Customer Security Programme evidence for physical and logical access correlation for swift operators ahead of independent assessment.

## Value

Gives the responsible team auditor-ready evidence for SWIFT CSP compliance by automating regulatory compliance evidence using indexes pacs, swift and sourcetypes badge:swipe, swift:operator:login. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=pacs sourcetype="badge:swipe" building="SWIFT_OPS" earliest=-24h
| join type=left user [| search index=swift sourcetype="swift:operator:login" earliest=-24d | stats latest(_time) as last_swift by user]
| eval gap=now()-last_swift
| where gap>7776000
| table user, last_swift
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Authentication.Authentication by Authentication.action, Authentication.user, Authentication.src | sort - count
```

## Implementation

(1) Segment SWIFT CSP evidence into dedicated indexes with strict role-based access. (2) Join operators to HR screening and vendor risk lookups refreshed weekly. (3) Validate MFA and PAM sourcetypes are CIM-mapped for Authentication models. (4) Retain evidence for at least 13 months aligned to attestation cycles. Indexes required: `index=pacs`, `index=swift`.

## Visualization

Table (operators), Single value (sessions), Bar chart (CSCF controls). Key fields: `gap`, `last_swift`, `user`, `user]`.

## Compliance

- **SWIFT CSP** CSCF v2025: 6.4 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for 6.4.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
