# UC-22.5.21 · MiFID II Best Execution — Venue Quality of Execution Report Ingestion Completeness

## Description

RTS 27/28 obligations produce periodic reports. Detecting missing files per venue or late uploads demonstrates control over the reporting supply chain used to evidence best execution monitoring.

## Value

RTS 27/28 obligations produce periodic reports. Detecting missing files per venue or late uploads demonstrates control over the reporting supply chain used to evidence best execution monitoring.

## Implementation

(1) Adjust `+35d` to regulatory calendar; (2) alert compliance operations; (3) vendor SLA tracking; (4) store checksums; (5) map to annual best ex board review.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=compliance sourcetype="qoe:file_manifest" earliest=-400d
| stats latest(received_ts) as last_recv by venue, report_month
| eval expected=strptime(report_month,"%Y-%m")
| where isnull(last_recv) OR last_recv > relative_time(expected,"+35d@d")
| table venue, report_month, last_recv
```
