# UC-22.29.29 · Consent record retention compliance

## Description

Provides APAC privacy supervisory evidence for consent record retention compliance aligned with PIPL; PDPA.

## Value

Provides APAC privacy supervisory evidence for consent record retention compliance aligned with PIPL; PDPA.

## Implementation

(1) Maintain `apac_transfer_register.csv` and consent or DPIA lookups referenced in runbooks. (2) Map technical identifiers to records of processing activities with legal review. (3) Restrict indexes to regional privacy roles with least privilege. (4) Quarterly DPIA and consent linkage review with local counsel.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=cmp sourcetype="consent:archive" earliest=-730d
| stats min(_time) as first, max(_time) as last by consent_id
| eval retained_days=round((last-first)/86400,0)
| where retained_days>2555
```
