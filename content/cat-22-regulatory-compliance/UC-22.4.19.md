# UC-22.4.19 · CCPA/CPRA — Dark Patterns — Forced Navigation Loops on “Do Not Sell or Share” Choice Screen

## Description

Dark patterns undermine consumer choice. Session paths that repeatedly bounce users away from the opt-out confirmation page without completion suggest manipulative UX—relevant to CPRA enforcement priorities on ease of exercise of rights.

## Value

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=web sourcetype="ux:privacy_flow" earliest=-30d
| sort 0 session_id, _time
| streamstats window=1 global=f last(step_name) as prev_step by session_id
| eval backtrack=if(step_name="dns_intro" AND prev_step="dns_confirm",1,0)
| stats sum(backtrack) as loop_count by session_id
| where loop_count>3
| sort - loop_count
```
