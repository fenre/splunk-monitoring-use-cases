# 22.39.4 — Cross-regulator consistency — divergent material facts across submissions

**Criticality:** high | **Difficulty:** advanced | **Status:** draft

## Description

Reads the structured material-fact fields of every regulator submission for an incident and alerts when the same incident has been described with divergent numbers or data classes. A diverging narrative across regulators is a pattern that escalates enforcement severity.

## Value

Prevents the 'regulator cross-reference' problem that has escalated several prominent supervisory cases from technical fines to broader corporate sanction.

## SPL

```spl
index=notifications sourcetype=regulator:submission earliest=-30d
| stats values(affected_subjects) as subjects_set values(data_classes) as data_set dc(affected_subjects) as subjects_uniques dc(data_classes) as data_uniques by incident_id
| where subjects_uniques>1 OR data_uniques>1
| table incident_id subjects_set data_set
```

## Implementation

(1) Normalise submission events to carry canonical material-fact fields; (2) run 24h after any new submission for the incident_id; (3) alert DPO + legal if divergence found. Indexes required: `index=notifications`; Sourcetypes: `sourcetype=regulator:submission`.

## Visualization

Table of divergent incidents, bar chart of divergence magnitude, single-value 'incidents with consistent submissions'. Key fields: `data_set`, `data_uniques`, `incident_id`, `subjects_set`, `subjects_uniques`.

## Compliance

- **GDPR** 2016/679: Art.33(3) (partial) — Art.33(3) requires specific material facts; divergence across submissions can be challenged.
- **NIS2** Directive (EU) 2022/2555: Art.23(4) (partial) — NIS2 requires a final report with same material facts; inconsistency is a deficiency.
- **DORA** Regulation (EU) 2022/2554: Art.19 (partial) — DORA Art.19 initial and final reports must be internally consistent.
- **UK GDPR** post-Brexit: Art.33(3) (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.33(3).

## Known False Positives

Genuine refinement of scope between the initial 72h notice and the 30-day follow-up is legitimate and expected; parametrise a tolerance window.
