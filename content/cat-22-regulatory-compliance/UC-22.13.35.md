# 22.13.35 — Default, Built-In, or Generic Account Usage on BES Hosts

**Criticality:** critical | **Difficulty:** intermediate | **Status:** N/A

## Description

We flag logons using default or generic accounts that should be disabled or renamed per system hardening standards.

## Value

Gives the responsible team auditor-ready evidence for NERC CIP compliance by automating regulatory compliance evidence using indexes wineventlog and sourcetypes WinEventLog:Security. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=wineventlog sourcetype="WinEventLog:Security" EventCode=4624 earliest=-7d
| lookup bes_cyber_asset_inventory.csv ComputerName AS host OUTPUT bes_system
| where isnotnull(bes_system)
| lookup generic_accounts_blocklist.csv Account_Name OUTPUT blocked_account
| where isnotnull(blocked_account)
| stats count by Account_Name, ComputerName, bes_system
| sort - count
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Authentication.Authentication by Authentication.action, Authentication.user, Authentication.src | sort - count
```

## Implementation

(1) Populate blocklist with Administrator, Guest, vendor defaults. (2) Tune for renamed admin accounts via SID lookup if available. (3) Page IR on hits. (4) Document approved break-glass in lookup. Indexes required: `index=wineventlog`; Sourcetypes: `sourcetype=WinEventLog:Security`.

## Visualization

Table (violations), Single value (event count), Bar chart (by account). Key fields: `Account_Name`, `ComputerName`, `bes_system`.

## Compliance

- **NERC CIP** current: CIP-007-6 R5 Part 5.2 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for CIP-007-6 R5 Part 5.2.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
