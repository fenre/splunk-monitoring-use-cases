# UC-22.11.36 · Pull-Request and Code Review Evidence for Payment Microservices

## Description

Ingests Git platform webhooks to verify merges to payment repositories include required reviewers, supporting secure software engineering practices for custom payment software.

## Value

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=devsecops sourcetype="github:pull_request" earliest=-7d
| lookup pci_payment_repos.csv repo OUTPUT pci_repo
| where pci_repo="true" AND merged="true"
| eval review_ok=if(review_count>=1 AND match(labels,"(?i)security-reviewed"),1,0)
| where review_ok=0
| table _time, repo, user, review_count, labels, url
| sort _time
```
