# UC-22.48.4 · Financial SoD — same identity posts AND approves a journal entry

## Description

Direct SOX ITGC control: manual journal entries must be posted and approved by distinct identities.

## Value

Provides continuous evidence of a universally-audited SOX control.

## Implementation

(1) Ensure ERP emits both events; (2) run hourly; (3) on fire, escalate to CFO + controller. Indexes required: `index=erp`; Sourcetypes: `sourcetype=oracle:gl`, `sourcetype=sap:gl`, `sourcetype=workday:gl`.

## Known False Positives

System-generated entries (automated accruals) post under a service account that must not be counted; filter by account type.
