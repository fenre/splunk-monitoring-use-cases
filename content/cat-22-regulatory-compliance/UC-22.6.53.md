# UC-22.6.53 · ISO/IEC 27001:2022 Clause 7.2 — Competence evidence: role-based training completion

## Description

Joins the role-to-module matrix against LMS completions to report each person's coverage. Clause 7.2 requires the organisation to determine and maintain competence.

## Value

Replaces the annual LMS export with a daily compliance signal. HR can notify managers proactively rather than after the audit finding.

## Implementation

(1) Onboard LMS completions; (2) maintain iso_27001_training_matrix.csv per role; (3) schedule UC daily; (4) compliance_pct<100 triggers an HR email; (5) quarterly report to ISMS committee.

Step 1 — Configure data collection

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=hr sourcetype=hr:role_assignment earliest=-30d
| stats latest(role) AS role BY user
| lookup iso_27001_training_matrix.csv role OUTPUT required_modules
| eval required_modules=split(required_modules,",")
| join user
  [search index=hr sourcetype=lms:completion status=completed earliest=-365d
   | stats values(module) AS completed_modules BY user]
| eval covered=mvmap(required_modules, if(isnotnull(mvfind(completed_modules,x)),1,0))
| eval compliance_pct=round(100*sum(covered)/mvcount(required_modules),0)
| where compliance_pct<100
| table user role required_modules completed_modules compliance_pct
```
