# UC-22.13.25 · Visitor Badge Without Escort Within Policy Window

## Description

We find visitor badge reads that lack a proximate escort badge swipe, supporting visitor control procedures.

## Value

We find visitor badge reads that lack a proximate escort badge swipe, supporting visitor control procedures.

## Implementation

(1) Fix join logic per PACS export (example simplified). (2) Use `transaction` on reader_id if vendor provides visit_id. (3) Daily report to reception. (4) Tune window to site policy.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=physical sourcetype=pacs:access badge_type="visitor" earliest=-7d
| where isnull(escort_badge_id) OR escort_present="false"
| lookup cip_psp_register reader_id OUTPUT psp_name
| stats count by visitor_id, reader_id, psp_name
| where count>=1
| table visitor_id, reader_id, psp_name, count
```
