# 22.13.68 — Transmission Station Threat Assessment Evidence Indexing

**Criticality:** high | **Difficulty:** intermediate | **Status:** N/A

## Description

We index completed threat assessment metadata for transmission stations so evidence dates and reviewers are searchable for CIP-014 reviews.

## Value

Gives the responsible team auditor-ready evidence for NERC CIP compliance by automating regulatory compliance evidence using indexes grc and sourcetypes cip014_threat_assessment, cip014_threat_assessment`. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=grc sourcetype=cip014_threat_assessment earliest=-730d
| eval completed=strptime(completed_date,"%Y-%m-%d")
| eval due_review=relative_time(completed,"+3y@d")
| where now()>due_review
| table station_id, completed_date, reviewer, due_review, next_assessment_scheduled
```

## Implementation

(1) Push assessment completions via HEC when signed. (2) Set review cadence per program (example three years). (3) Alert owners before due. (4) Attach PDF hash externally if required. Indexes required: `index=grc`; Sourcetypes: `sourcetype=cip014_threat_assessment`, `sourcetype=cip014_threat_assessment``.

## Visualization

Table (stations due), Timeline, Single value (count). Key fields: `completed`, `completed_date`, `due_review`, `next_assessment_scheduled`, `reviewer`, `station_id`.

## Compliance

- **NERC CIP** current: CIP-014-3 R1 Part 1.1 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for CIP-014-3 R1 Part 1.1.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
