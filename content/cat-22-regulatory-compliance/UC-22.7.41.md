# UC-22.7.41 · Root Cause Field Completeness on Closed Incidents

## Description

Ensures analysts filled root-cause codes when closing incidents for after-action analysis quality.

## Value

Ensures analysts filled root-cause codes when closing incidents for after-action analysis quality.

## Implementation

(1) Make field mandatory in workflow; (2) train SOC; (3) export for RC.IM.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: ServiceNow.
• Ensure the following data sources are available: `snow:incident`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Make field mandatory in workflow; (2) train SOC; (3) export for RC.IM.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=itsm sourcetype=snow:incident earliest=-90d state=7
| where isnull(u_root_cause_code) OR u_root_cause_code=""
| stats count by assignment_group
```
