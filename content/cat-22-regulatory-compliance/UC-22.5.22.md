# UC-22.5.22 · MiFID II Best Execution — Slippage vs Reference Price by Client Segment and Instrument Class

## Description

Best execution is measured by outcomes. Trending slippage against a consolidated reference price by retail vs professional clients evidences differentiated quality monitoring required for policy tuning and disclosures.

## Value

Gives the responsible team auditor-ready evidence for MiFID II compliance by automating regulatory compliance evidence using indexes trading and sourcetypes ems:fill. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Define `ref_price` methodology consistently; (2) exclude auctions; (3) monthly best ex committee pack; (4) integrate with order routing analysis; (5) document outliers investigated. Indexes required: `index=trading`; Sourcetypes: `sourcetype=ems:fill`.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: OMS fills + market data snapshot service.
• Ensure the following data sources are available: `index=trading` `sourcetype="ems:fill"` (client_segment, instrument_class, fill_ts, price, ref_price, symbol).
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Define `ref_price` methodology consistently; (2) exclude auctions; (3) monthly best ex committee pack; (4) integrate with order routing analysis; (5) document outliers investigated.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=trading sourcetype="ems:fill" earliest=-30d
| eval slip_bps=abs(price-ref_price)/ref_price*10000
| stats median(slip_bps) as med_slip, perc95(slip_bps) as p95_slip by client_segment, instrument_class, symbol
| where client_segment="retail" AND p95_slip>50
| sort - p95_slip
```
