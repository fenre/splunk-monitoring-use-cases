# UC-22.8.27 · SOC 2 A1 — Capacity Planning Signals for In-Scope Production Services

## Description

A1 addresses availability commitments. Forecasting CPU, memory, queue depth, or connection pool utilization against committed headroom demonstrates proactive capacity management—not reactive firefighting after outages.

## Value

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=metrics metric_name="cpu.util" entity=prod_payments_* earliest=-90d
| timechart span=1d avg(_value) as avg_cpu
| predict avg_cpu as forecast algorithm=LLP future_timespan=14 period=7
| where forecast>85
| table _time, avg_cpu, forecast
```
