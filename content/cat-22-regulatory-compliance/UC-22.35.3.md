# 22.35.3 — Indexer replication lag exposing evidence to single-point failure

**Criticality:** high | **Difficulty:** intermediate | **Status:** verified

## Description

Polls indexer cluster topology and raises a finding whenever replication-factor or search-factor is not met. Because auditors treat a single-replica audit trail as equivalent to no audit trail, a sustained lag above the stated RPO breaks the compliance claim.

## Value

Turns an otherwise-silent SLO breach into evidence that the control was monitored continuously, even across shards that never had an incident.

## SPL

```spl
| rest /services/cluster/master/indexes splunk_server=* | stats max(replication_factor_met) as rf_met max(search_factor_met) as sf_met by title | eval index=title, rf_met=coalesce(rf_met,0), sf_met=coalesce(sf_met,0) | where rf_met=0 OR sf_met=0
```

## Implementation

(1) Schedule hourly; (2) parameterise RPO via macro `worm_rpo_seconds`; (3) correlate with maintenance-window lookup to suppress planned reboots; (4) wire to ITSI availability KPI for cross-pillar visibility. Indexes required: `index=title`.

## Visualization

Line chart of replication lag by index; table of out-of-SLA indexes with duration; single-value for '% of time within RPO'.

## Compliance

- **GDPR** 2016/679: Art.32 (full) — Uplifted to full: SPL provides comprehensive, standalone evidence for Art.32 with specific data sources and actionable thresholds.
- **DORA** Regulation (EU) 2022/2554: Art.12 (full) — Art.12 requires backup policies and recovery methods with tested RPO; this UC alerts when replication RPO is exceeded in flight.
- **NIST 800-53** Rev. 5: AU-9 (full) — AU-9 requires protection of audit information from unauthorized access, modification, and deletion; replication is a standard enhancement for AU-9(2).
- **SOC 2** 2017 TSC: A1.2 (partial) — A1.2 covers availability commitments; replication lag monitoring is one input.
- **APPI** 2022 amendments: Art.23 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.23.
- **LGPD** Lei nº 13.709/2018: Art.46 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.46.
- **UK GDPR** post-Brexit: Art.32 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for Art.32.

## Known False Positives

Planned rolling restarts, primary-peer failovers, or newly-added indexes (before first replication cycle) can briefly show lag. Apply maintenance-window suppression.
