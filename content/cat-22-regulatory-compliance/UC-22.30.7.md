# 22.30.7 — Outsourcing arrangements risk monitoring

**Criticality:** critical | **Difficulty:** advanced | **Status:** N/A

## Description

Supports MAS TRM supervisory examination readiness for outsourcing arrangements risk monitoring.

## Value

Gives the responsible team auditor-ready evidence for MAS TRM compliance by automating regulatory compliance evidence using indexes vendor and sourcetypes mas:outsourcing. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=vendor sourcetype="mas:outsourcing" earliest=-180d
| stats latest(risk_rating) as rr by vendor_name
| where rr IN ("High","Severe")
```

## CIM SPL

```spl
| tstats summariesonly=t latest(All_Changes.status) as agg_value from datamodel=Change.All_Changes by All_Changes.action, All_Changes.object_category, All_Changes.user | sort - agg_value
```

## Implementation

(1) Maintain `apac_material_services.csv` for important business services and regulator tags. (2) Send weekly digests to CRO and technology risk committees. (3) Validate DB Connect concurrency for audit peak windows. (4) Archive results to a restricted summary index for retention governance. Indexes required: `index=vendor`; Sourcetypes: `sourcetype=mas:outsourcing`.

## Visualization

Column chart (events by regulator), Table (privileged actions), Timeline (notification milestones). Key fields: `rr`, `vendor_name`.

## Compliance

- **MAS TRM** 2021: §4.1.1 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for §4.1.1.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
