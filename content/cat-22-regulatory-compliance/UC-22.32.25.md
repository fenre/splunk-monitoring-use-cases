# 22.32.25 — CJI media protection and transfer monitoring

**Criticality:** high | **Difficulty:** expert | **Status:** N/A

## Description

Supports CJIS Security Policy compliance evidence for cji media protection and transfer monitoring in Splunk dashboards and scheduled reports.

## Value

Gives the responsible team auditor-ready evidence for CJIS compliance by automating regulatory compliance evidence using indexes removable and sourcetypes dlp:usb, usb:mount. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=removable sourcetype="usb:mount" OR sourcetype="dlp:usb" earliest=-30d
| search tag=cjis OR host="CJIS-*"
| stats count by user, device_serial
| sort - count
```

## CIM SPL

```spl
| tstats summariesonly=t count from datamodel=Authentication.Authentication by Authentication.user | sort - count
```

## Implementation

(1) Maintain `amer_authorization_boundary.csv` with ATO, CUI, and CJIS scope flags. (2) Integrate POA&M exports via scheduled output and lookup refresh for ISSM workflows. (3) Separate LGPD and federal datasets with index- and role-level controls. (4) Quarterly control mapping workshop with ISSM, ISO, and privacy counsel. Indexes required: `index=removable`; Sourcetypes: `sourcetype=dlp:usb`, `sourcetype=usb:mount`.
