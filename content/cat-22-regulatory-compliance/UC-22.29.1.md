Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=db_audit sourcetype="oracle:audit" Action="SELECT" earliest=-36h
| stats count by DBUserName, Obj_Name, client_ip
| iplocation client_ip
| where Country!="China" AND match(Obj_Name,"(?i)resident")
| sort - count
```
