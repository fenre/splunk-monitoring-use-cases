# UC-22.15.10 · Vendor Remote Maintenance Window and Dual-Control Evidence

## Description

Monitors vendor VPN and PSM sessions against contracted hours and approval records to evidence supplier risk management for OT maintenance.

## Value

Gives the responsible team auditor-ready evidence for IEC 62443 compliance by automating regulatory compliance evidence using indexes cyberark, vpn and sourcetypes cyberark:psm, pan:globalprotect. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Populate vendor contract lookup; (2) force vendor IDs through PSM; (3) alert security + procurement on violations; (4) quarterly supplier review export. Indexes required: `index=cyberark`, `index=vpn`; Sourcetypes: `sourcetype=cyberark:psm`, `sourcetype=pan:globalprotect`.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Add-on for Palo Alto Networks (2757), Splunk Add-on for CyberArk (4295).
• Ensure the following data sources are available: `pan:globalprotect` or `pan:sslvpn`; `cyberark:psm`.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Populate vendor contract lookup; (2) force vendor IDs through PSM; (3) alert security + procurement on violations; (4) quarterly supplier review export.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=vpn sourcetype="pan:globalprotect" earliest=-14d
| lookup vendor_access.csv user OUTPUT vendor, window
| eval hod=strftime(_time,"%H")
| eval off_window=case(window="business" AND (hod<7 OR hod>18),1, window="24x7",0, true(),0)
| join type=left user [ search index=cyberark sourcetype="cyberark:psm" earliest=-14d | stats count as psm_hits by user ]
| where vendor!="Internal" AND (off_window=1 OR psm_hits=0)
| stats count by vendor, user, off_window
```
