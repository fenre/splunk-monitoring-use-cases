# UC-22.6.11 · Threat Intelligence Feed Freshness and STIX Object Ingest Gaps

## Description

Measures age of threat intel objects and ingest errors so external threat information supports decisions per Annex A.

## Value

Measures age of threat intel objects and ingest errors so external threat information supports decisions per Annex A.

## Implementation

(1) Align with TI subscription SLAs; (2) alert stale feeds; (3) document in threat intel procedure.

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Enterprise Security (Splunkbase 263) or custom HEC.
• Ensure the following data sources are available: `threat_intel_domain.csv` lookup.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Align with TI subscription SLAs; (2) alert stale feeds; (3) document in threat intel procedure.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
| inputlookup threat_intel_domain.csv
| eval age_days=(now()-strptime(last_modified,"%Y-%m-%d"))/86400
| where age_days>30
| stats count by source, age_days
```
