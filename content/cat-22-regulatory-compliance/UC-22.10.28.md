# UC-22.10.28 ¬∑ Workstation Security ‚Äî Unattended Unlocked Sessions in Clinical Pods

## Description

Correlates long gaps between workstation unlock events in nursing stations with active Hyperspace sessions to reduce shoulder-surfing and unauthorized viewing risks.

Step 2 ‚Äî Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=windows sourcetype="WinEventLog:Security" EventCode IN ("4800","4801") earliest=-7d
| eval action=if(EventCode="4800","lock","unlock")
| transaction WorkstationName startswith=eval(action="unlock") endswith=eval(action="lock") maxspan=8h
| eval idle_during=if(duration>3600 AND eventcount<3,1,0)
| where idle_during=1
| table WorkstationName, duration, user, _time
```
