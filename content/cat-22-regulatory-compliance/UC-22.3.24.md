# UC-22.3.24 ¬∑ DORA ICT Concentration ‚Äî Substitutability and Secondary Sourcing Readiness Index

## Description

Concentration risk mitigates when substitutable alternatives exist. Scoring each critical provider on contract portability, data export tested, and runbook completeness gives a defensible substitutability index for the register of information.

## Value

Gives the responsible team auditor-ready evidence for DORA compliance by automating regulatory compliance evidence. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Calibrate weights with risk team; (2) refresh after DR tests; (3) link `provider_id` to RoI; (4) annual reassessment workflow; (5) do not embed legal advice‚Äîstore lawyer sign-off reference externally. Lookups: `dora_substitutability`, `dora_substitutability.csv`.

Step 2 ‚Äî Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
| inputlookup dora_substitutability.csv
| eval last_test=strptime(last_export_test_date,"%Y-%m-%d")
| eval test_age_days=round((now()-last_test)/86400,0)
| eval readiness=contract_portable*0.4 + if(test_age_days<180,0.3,0) + runbook_score*0.3
| where readiness<0.6
| table provider_id, readiness, contract_portable, test_age_days, runbook_score
| sort readiness
```
