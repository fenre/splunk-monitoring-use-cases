# UC-22.14.43 · Memory Protection Signals — Exploitation Primitives in EDR Telemetry

## Description

Uses EDR or Sysmon events indicating process hollowing or suspicious memory protections for SI-16 memory protection.

## Value

Uses EDR or Sysmon events indicating process hollowing or suspicious memory protections for SI-16 memory protection.

## Implementation

(1) Tune for developer VMs; (2) enrich with parent-child chain 1/3; (3) alert on LSASS-targeting patterns; (4) integrate with IR-4 containment playbooks.

| sort - count
