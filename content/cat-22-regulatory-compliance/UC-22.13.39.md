# UC-22.13.39 ┬Ę Reportable Incident Classification ŌĆö BES Reliability Impact Signals

## Description

We correlate OT historian or EMS quality flags with cyber alerts to help classify incidents that may affect BES reliability or operations.

## Value

We correlate OT historian or EMS quality flags with cyber alerts to help classify incidents that may affect BES reliability or operations.

## Implementation

(1) Normalize substation keys between historians and security tools. (2) Train analysts on EOP impact criteria. (3) Document classification decisions in ITSM. (4) Retain historian extracts per legal hold guidance.

Step 1 ŌĆö Configure data collection

Step 2 ŌĆö Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=ot_hist sourcetype=scada:point earliest=-24h (quality="BAD" OR quality="UNCERTAIN")
| bin _time span=15m
| stats count by substation, point_name, _time
| join type=left substation [
    search index=ot_security sourcetype="ot:alert" earliest=-24h
    | stats latest(description) as cyber_alert by substation
  ]
| where isnotnull(cyber_alert)
| table _time, substation, point_name, count, cyber_alert
```
