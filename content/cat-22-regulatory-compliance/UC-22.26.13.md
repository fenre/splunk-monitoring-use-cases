# UC-22.26.13 · PSA compliance evidence

## Description

Packages access reviews, alarm management changes, and temporary bypasses for PSA inspection readiness.

## Value

Gives the responsible team auditor-ready evidence for NO Petroleumsforskriften compliance by automating regulatory compliance evidence using indexes app and sourcetypes eptw:audit. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## Implementation

(1) Maintain supporting lookups (classified hosts, sectors, transfers) aligned to Norwegian regulatory language. (2) Schedule searches with Europe/Oslo time semantics for reporting cutoffs. (3) Route ES notables to the correct national security, OT, or privacy queues. (4) Quarterly false-positive tuning with HR, operations, and legal stakeholders. Indexes required: `index=app`; Sourcetypes: `sourcetype=eptw:audit`.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=app sourcetype="eptw:audit" site="NOR-OIL-*" earliest=-30d
| transaction permit_id maxspan=2h maxpause=10m
| eval mfa_ok=if(mvfind(_raw,"mfa_result=success")>=0,1,0)
| stats min(mfa_ok) as chain_ok by permit_id
| where chain_ok=0
| table permit_id
```
