# 22.15.48 — Cross-Zone Protocol Anomaly Detection — IEC 61850 GOOSE Floods

**Criticality:** critical | **Difficulty:** advanced | **Status:** N/A

## Description

Spots unusual GOOSE/AppID volumes between substation zones indicating mis-wiring, VLAN bleed, or malicious injection.

## Value

Gives the responsible team auditor-ready evidence for IEC 62443 compliance by automating regulatory compliance evidence using indexes ot_sub and sourcetypes iec61850:goose. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=ot_sub sourcetype="iec61850:goose" earliest=-24h
| stats count by app_id, src_mac, vlan, bay
| eventstats median(count) as med by app_id
| eval ratio=round(count/med,2)
| where ratio>5 OR count>10000
| sort - ratio
```

## Implementation

(1) Baseline per bay during commissioning; (2) segregate process bus; (3) integrate relay vendor validation; (4) NERC/IEC dual use where applicable. Indexes required: `index=ot_sub`; Sourcetypes: `sourcetype=iec61850:goose`.

## Visualization

Time chart, Table, Single value (anomalous APPIDs). Key fields: `app_id`, `bay`, `ratio`, `src_mac`, `vlan`.

## Compliance

- **IEC 62443** 2013-ongoing: FR 6.2 (partial) — Uplifted from contributing to partial: UC provides meaningful monitoring evidence for FR 6.2.

## Known False Positives

Administrative tasks, scheduled jobs or platform updates can match this pattern — correlate with change management, maintenance windows and user role before raising severity.
