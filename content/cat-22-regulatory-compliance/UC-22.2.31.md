# UC-22.2.31 · NIS2 Entity Classification Validation (Essential vs Important)

## Description

Incorrect classification drives wrong reporting deadlines and control expectations. This use case reconciles legal entity classification in `nis2_entity_classification.csv` with CMDB ownership, revenue sector tags, and national registry identifiers — flagging assets attached to misclassified entities.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
| inputlookup nis2_entity_classification.csv
| join legal_entity_id [
    search index=cmdb earliest=-1d
    | stats values(nis2_entity_tag) as cmdb_tags by legal_entity_id
]
| eval mismatch=if(NOT match(mvjoin(cmdb_tags,"|"), entity_tier), 1, 0)
| where mismatch=1 OR isnull(cmdb_tags)
| table legal_entity_id, entity_tier, cmdb_tags, mismatch
```
