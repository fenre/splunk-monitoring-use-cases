# UC-22.5.14 · MiFID II Client Suitability — Investment Objective Mismatch Alerts vs Held Positions

## Description

Ongoing suitability requires consistency between stated objectives (income vs growth) and position risk. Detecting concentrated speculative exposures for conservative profiles supports conduct supervision and product governance linkage.

## Value

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=wealth sourcetype="pm:position" earliest=-1d
| stats sum(notional) as exp, sum(eval(if(asset_class="derivatives",notional,0))) as deriv_exp by client_id
| lookup crm_profile_objective.csv client_id OUTPUT objective_tier
| eval deriv_ratio=if(exp>0, deriv_exp/exp, 0)
| where objective_tier="conservative" AND deriv_ratio>0.2
| table client_id, objective_tier, deriv_ratio, exp
```
