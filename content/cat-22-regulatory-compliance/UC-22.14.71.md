# UC-22.14.71 · Cryptographic Protection — BitLocker or Disk Encryption Status Drops

## Description

Tracks endpoint encryption compliance for SC-13 cryptographic protection.

## Value

## Detailed Implementation

Prerequisites
• Install and configure the required add-on or app: Splunk Add-on for Microsoft Windows (Splunkbase 742), Intune HEC.
• Ensure the following data sources are available: `WinEventLog:Microsoft-Windows-BitLocker/Operational` or inventory JSON.
• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=os sourcetype=win:bitlocker_status earliest=-24h protection_status="Off"
| stats count by host, volume, reason
| sort - count
```
