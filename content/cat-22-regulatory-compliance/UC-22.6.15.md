# UC-22.6.15 · ICT Readiness for BC — Backup Window Overruns vs RPO

## Description

Compares backup job duration and completion time against RPO windows for ICT supporting business continuity.

## Value

Compares backup job duration and completion time against RPO windows for ICT supporting business continuity.

## Implementation

• For app installation, inputs.conf, and Splunk directory layout, see the Implementation guide: docs/implementation-guide.md

Step 1 — Configure data collection
(1) Normalize duration fields; (2) tune for incremental chains; (3) feed BC tests.

Step 2 — Create the search and alert
Run the following SPL in Search (then save as report or alert; adjust time range and threshold as needed):

```spl
index=backup sourcetype=netbackup:job earliest=-14d
| lookup app_rpo_hours.csv policy_name OUTPUT rpo_hours
| eval end_epoch=_time+duration_sec
| eval overrun=if((end_epoch-_time)/3600>rpo_hours,1,0)
| where status!="success" OR overrun=1
| stats count by policy_name, client_name
```
