# 22.13.69 — Transmission Physical Security Plan Control Checklist Status

**Criticality:** high | **Difficulty:** intermediate | **Status:** N/A

## Description

We track periodic checklist completions for transmission station physical security plan controls such as lighting, fencing, and cameras.

## Value

Gives the responsible team auditor-ready evidence for NERC CIP compliance by automating regulatory compliance evidence using indexes grc and sourcetypes cip014_control_check, cip014_control_check`. Replaces manual evidence gathering with a continuous Splunk search that produces structured, time-stamped artefacts suitable for regulatory examination and internal audit review.

## SPL

```spl
index=grc sourcetype=cip014_control_check earliest=-400d
| eval performed=strptime(performed_date,"%Y-%m-%d")
| eval overdue=if(now()-performed>7776000,1,0)
| where overdue=1 AND result!="pass"
| table station_id, control_id, performed_date, result, owner
```

## Implementation

(1) Encode ninety-day inspection default (tune per plan). (2) Mobile form integration to Splunk via HEC. (3) Escalate failed controls to field ops. (4) Photo evidence stored outside Splunk with URL field. Indexes required: `index=grc`; Sourcetypes: `sourcetype=cip014_control_check`, `sourcetype=cip014_control_check``.
